## 1. 系统简介

「智慧宠物烟感安全系统」是一个以烟感/粉尘监测为核心、围绕鹦鹉（及其他小型宠物）笼舍环境安全的综合照护平台。它通过传感器实时采集笼舍的烟雾/粉尘浓度、温度、湿度，自动判断风险等级，在超标时触发告警并联动排风扇、报警灯、蜂鸣器、空气净化器等设备保护宠物；同时提供宠物档案、成长报告、医疗助手、记账本、饲养手册等一站式照护能力。

**核心价值**：

- **安全守护**：24 小时监测笼舍空气环境，异常秒级告警，避免特氟龙、香烟、香薰等致命气体在主人不在时伤害鹦鹉。
- **环境可视化**：温湿度、粉尘浓度仪表盘直观呈现，让主人一眼看懂笼舍是否舒适。
- **成长记录**：自动归档监控截图、体重曲线、健康评分，形成完整的宠物成长档案。
- **照护辅助**：智能问诊、附近医院、食物安全查询、记账本，覆盖日常饲养全场景。

## 2. 首页布局与导航

首页采用三栏式布局，以当前选中的鹦鹉为中心：

```
┌─────────────────────────────────────────────────────┐
│  左栏            │    中栏              │  右栏      │
│                  │                      │            │
│  宠物档案入口     │  当前鹦鹉 + 切换      │  用户设置   │
│  成长报告入口     │  实时视频监控         │  医疗助手   │
│                  │  记账本入口           │  饲养手册   │
└─────────────────────────────────────────────────────┘
```

- **当前鹦鹉卡片**：显示当前选中的鹦鹉头像与名称，点击右侧下拉按钮可切换到其他鹦鹉，切换后所有模块（监控、成长报告、记账）都会跟随切换。
- **入口卡片**：点击任意入口卡片进入对应详情页，详情页左上角「返回」按钮可回到首页。

## 3. 实时监控模块

实时监控是系统的核心模块，位于首页中央。

### 3.1 实时视频通话

- **进入实时模式**：点击监控卡片的「打开实时监控」按钮，进入实时视频通话模式。
- **视频画面**：当前为模拟实时画面（后续将接入真实摄像头 WebRTC/HLS 流）。
- **环境指标侧栏**：视频左侧展示温度、湿度、粉尘浓度三个仪表盘，数据每 3 秒从后端 `/api/smoke/realtime` 拉取一次。
- **工具栏（右下）**：
  - **截图**：将当前画面截图保存到宠物成长相册（保存到本地，后续可上传到 `pet_media_record`）。
  - **近一周记录**：查看该鹦鹉关联设备的近期告警记录。
  - **麦克风开关**：开启/关闭麦克风（联调中）。
  - **音量调节**：调节监听音量。
  - **全屏**：放大监控画面。
- **退出**：点击左上角返回按钮退出实时模式。

### 3.2 环境指标仪表盘

点击任一环境指标（温度/湿度/粉尘）可弹出仪表盘详情：

| 指标 | 单位 | 仪表盘量程 | 等级判定 |
|---|---|---|---|
| 温度 | ℃ | 0–45 | <18 偏低 / 18–30 适宜 / >30 偏高 |
| 湿度 | %RH | 0–100 | <40 偏低 / 40–70 适宜 / >70 偏高 |
| 粉尘浓度 | μg/m³ | 0–120 | <35 低 / 35–80 中 / >80 高 |

> 仪表盘下方标注「已连接后端实时数据」或「后端未连接，当前为保底模拟值」，便于用户判断数据可信度。

## 4. 宠物档案管理

### 4.1 查看档案列表

从首页左栏「宠物档案」入口进入，可看到当前用户名下所有鹦鹉档案卡片，展示头像、年龄阶段、名称、品种、出生日期、体重、性别、当前状态。

### 4.2 新增档案

1. 点击页面顶部「增加档案」按钮。
2. 在弹窗中填写：
   - **鹦鹉种类**：从下拉选择（虎皮、牡丹、玄凤、小太阳、和尚、吸蜜、凯克、折衷、灰鹦鹉等）。
   - **鹦鹉名字**：如「啾啾」。
   - **出生日期**：格式 `xxxx-xx-xx`，系统自动计算年龄阶段（幼年/青少年/成年/老年）。
   - **当前体重**：如 `78g`。
3. 点击「保存」，新档案加入列表并自动选为当前鹦鹉。

### 4.3 档案详情

点击任一档案卡片进入详情：

- **基本资料**：展示并支持「编辑」名称、品种、出生日期、体重、性别、当前状态。
- **体重记录**：折线图展示历次体重，点击可查看大图。
- **成长相册**：展示已归档的照片/截图。
- **录入体重**：在「录入体重」卡片中输入今日体重并保存，自动追加到体重曲线。

> 建议每周至少录入一次体重，体重持续下降是鹦鹉患病的早期重要信号。

## 5. 成长报告

从首页左栏「成长报告」入口进入。

### 5.1 报告周期切换

顶部切换「日报 / 周报 / 月报」：

- **日报**：按小时展示当日曲线。
- **周报**：按周一至周日展示。
- **月报**：按第 1–4 周展示。

### 5.2 关键指标卡片

展示健康评分、睡眠时长、鸣叫次数、进食次数、排泄次数及较上期变化趋势。

### 5.3 曲线区

- **温度曲线 / 湿度曲线 / 粉尘曲线 / 体重变化曲线**：点击任一曲线可查看大图。
- 温湿度/粉尘曲线为可点击的仪表盘卡片，点击弹出仪表盘详情。

### 5.4 记录区

- **照片记录**：最兴奋照片、睡觉照片等行为分类照片。
- **录音**：学舌练习、歌曲练习、鸣叫情绪录音。
- **健康风险提醒**：如「下午羽粉偏高，建议通风 20 分钟」。

> 成长报告由系统基于当日传感数据与人工录入行为汇总生成（`pet_daily_report` 表），周报/月报由日报聚合。

## 6. 医疗助手

从首页右栏「医疗助手」入口进入，包含三个子模块：

### 6.1 智能问诊

1. 填写「外在表现问卷」：
   - 精神状态：精神很好 / 精神一般 / 明显萎靡
   - 进食情况：正常进食 / 食量下降 / 拒食
   - 呼吸表现：无异常 / 偶尔张口 / 持续张口呼吸
   - 排泄情况：正常 / 偏稀 / 颜色异常
2. 点击「提交」，系统给出初步风险判断与处置建议。
3. **重要**：智能问诊仅供初步参考，不能替代专业兽医诊断。若出现「明显萎靡 + 拒食 + 持续张口呼吸」等危险组合，请立即就医，参见 [鹦鹉急救指南.md](鹦鹉急救指南.md)。

### 6.2 附近医院

- 展示可治疗异宠（鸟类）的医院地图与列表，含名称、地址、电话。
- 点击「刷新」可循环查看不同医院。
- 建议提前保存 1–2 家异宠医院联系方式作为应急储备。

### 6.3 病历

- 按时间记录就诊、用药、复查事项。
- 支持搜索（按关键字）、新增、修改。
- 建议每次就诊后及时记录，便于追溯病情变化。

## 8. 饲养手册

从首页右栏「饲养手册」入口进入，包含三个子模块：

### 8.1 教程库

- 新手喂养、剪羽、药浴、清洁等教程。
- 支持按关键字搜索。

### 8.2 食物安全

- 输入食物名称（如「苹果」），选择种类（蔬菜/水果/肉类/昆虫/谷物）。
- 点击「查询」获得可食用性结果与建议。
- **重要**：首次喂食任何新食物都应少量尝试，避开盐、糖、油和调味料。详细有毒食物清单见 [parrot-knowledge-base.md](parrot-knowledge-base.md) 第 3.2 节。

### 8.3 拍照识鸟

- 输入图片或线索（如「绿色身体、红色尾羽」）。
- 点击「识别」获得可能的鹦鹉种类与介绍。

## 10. 告警与联动设备

### 10.1 告警等级

系统根据环境数据自动判定风险等级：

| 烟雾/粉尘浓度（ppm） | 风险等级 | 页面主题 | 行为 |
|---|---|---|---|
| 0–100 | normal 正常 | 绿色 | 安全 |
| 101–199 | low 低风险 | 黄绿色 | 轻度提示 |
| 200–400 | medium 中风险 | 橙色 | 计入今日告警 |
| >400 | high 高风险 | 红色 | 触发危险主题 + 设备联动 |

### 10.2 自动联动

当风险达到中风险及以上，系统自动联动（可在 `device_control` 中配置 `auto_linkage`）：

- **排风扇**：开启排出受污染空气。
- **空气净化器**：开启过滤粉尘/羽粉。
- **报警灯 + 蜂鸣器**：声光提醒主人。

### 10.3 手动控制

用户可在设备控制面板手动开关任一联动设备。

### 10.4 告警处理

- 告警触发后，状态为 `pending`（待处理）。
- 主人现场处置后，在「告警日志 → 处理」中填写处理人、备注，状态变为 `resolved`。
- 详细应急流程见 [告警应急处理.md](告警应急处理.md)。

## 11. 环境指标解读

### 11.1 温度

- **适宜**：18–30℃（多数鹦鹉），折衷等雨林品种需 24–30℃。
- **过低（<18℃）**：鹦鹉代谢加快，长期可致免疫力下降。处置：加温（保温灯/加热垫），避免直吹。
- **过高（>30℃）**：张口呼吸、中暑风险。处置：开窗通风、避免空调直吹、提供洗澡水。

### 11.2 湿度

- **适宜**：40–70%（多数鹦鹉），折衷需 60–80%。
- **过低（<40%）**：羽毛脆裂、皮肤脱屑。处置：加湿器、喷雾。
- **过高（>70%）**：曲霉菌滋生风险。处置：除湿、通风。

### 11.3 粉尘浓度

- **低（<35 μg/m³）**：空气优良。
- **中（35–80 μg/m³）**：提示通风，关注羽粉来源。
- **高（>80 μg/m³）**：立即处置，参见 [告警应急处理.md](告警应急处理.md)。

> 各品种的具体适宜区间见 [parrot-knowledge-base.md](parrot-knowledge-base.md) 第 7 章。

## 12. 常见问题

**Q1：首页环境指标显示「后端未连接，当前为保底模拟值」怎么办？**
A：表示后端服务未启动或设备离线。请检查后端（8080 端口）是否运行，或确认 `device/getData` 是否正常订阅 MQTT。

**Q2：怎么设置合理的告警阈值？**
A：进入「系统设置 → 阈值配置」，默认 warning=200 ppm、danger=400 ppm。若饲养呼吸道敏感品种（灰鹦鹉、凤头、派翁尼斯），建议将粉尘 warning 下调至 25、danger 下调至 60 μg/m³。

**Q3：联动设备什么时候会自动开启？**
A：当风险等级达到 medium（200 ppm）及以上时，所有 `auto_linkage=1` 的设备自动开启。恢复正常后自动关闭。

**Q4：成长报告的「健康评分」怎么算的？**
A：基于当日环境数据（粉尘/温湿度是否在适宜区间）与行为指标（睡眠、进食、排泄）综合计算，0–100 分。环境异常会显著拉低评分。

**Q5：监控截图保存在哪里？**
A：当前保存在浏览器本地（localStorage），后续将上传到 `pet_media_record` 表，可在「宠物档案 → 成长相册」查看。

**Q6：可以同时养多只鹦鹉吗？**
A：可以。每只鹦鹉有独立档案，可绑定不同笼舍与设备。首页「当前鹦鹉」可切换，所有模块跟随切换。
