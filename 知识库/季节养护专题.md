# 鹦鹉季节养护专题

> 鹦鹉对环境变化敏感，季节更替时温差、湿度、光照、食物等因素变化，容易引发应激、换羽、繁殖、疾病。本专题按季节提供针对性养护建议，并结合本系统的环境监测能力，帮助主人提前预防季节性问题。

## 1. 春季养护（3–5 月）

### 1.1 季节特点

- 气温回升，但早晚温差大（可达 10℃以上）。
- 空气干燥转湿润，花粉、柳絮增多。
- 鹦鹉进入繁殖期与换羽准备期，激素活跃。
- 病原微生物开始活跃。

### 1.2 养护要点

| 方面 | 建议 |
|---|---|
| 温度 | 维持 20–26℃，避免早晚温差 >8℃ 触发系统 `temp_low` 告警 |
| 通风 | 增加通风频次，但避免冷风直吹；花粉季关注粉尘告警 |
| 光照 | 增加自然光照，促进换羽与激素平衡 |
| 饮食 | 增加新鲜蔬果（深绿、橙色蔬菜补维生素 A），补充钙质（产蛋期雌鸟）|
| 繁殖 | 雌鸟注意蛋难产风险，提供墨鱼骨；不打算繁殖的可减少筑巢刺激 |
| 体检 | 春季是体检好时机，建议年度鸟类兽医体检 |

### 1.3 春季常见问题

- **过敏/打喷嚏**：花粉增多，关注粉尘告警，必要时关窗、开净化器。
- **求偶攻击**：繁殖期雄性激素升高，可能出现攻击行为，避免过度刺激。
- **蛋难产**：雌鸟多发，参见 [鹦鹉急救指南.md](鹦鹉急救指南.md) 第 10 节。

## 2. 夏季养护（6–8 月）

### 2.1 季节特点

- 高温高湿，易中暑、食物变质。
- 空调使用频繁，温差与直吹问题突出。
- 蚊虫、寄生虫活跃。
- 鹦鹉饮水增加，洗澡需求增强。

### 2.2 养护要点

| 方面 | 建议 |
|---|---|
| 温度 | 控制在 28℃以下，>30℃ 触发 `temp_high` 告警需立即处置 |
| 湿度 | 梅雨季高湿需除湿，>70% 触发 `humidity_high` 告警 |
| 空调 | 可开空调，但**避免直吹**鹦鹉；外出时调至 26–28℃ |
| 饮水 | 每日更换 2 次以上，防止变质；提供洗澡水 |
| 食物 | 蔬果易变质，2 小时内未吃完需丢弃；颗粒粮防潮防虫 |
| 防虫 | 安装纱窗防蚊虫；定期检查体外寄生虫 |
| 洗澡 | 增加洗澡频次（每日 1 次可），洗澡后确保干燥 |

### 2.3 中暑预防与急救

- 笼舍避开阳光直射，拉窗帘。
- 高温日提供冰镇水瓶（放笼外辅助降温，不直接接触）。
- 出门前确认空调开启与设备在线。
- 中暑症状与急救见 [鹦鹉急救指南.md](鹦鹉急救指南.md) 第 4 节。

### 2.4 夏季食物保鲜

- 颗粒粮分小袋密封，放阴凉处。
- 自制软食（煮红薯、谷物粥）现做现吃，不隔夜。
- 水果切块后冷藏，喂食前回温。

## 3. 秋季养护（9–11 月）

### 3.1 季节特点

- 气温下降，温差加大。
- 鹦鹉进入换羽高峰期，营养需求增加。
- 部分品种准备越冬，食欲增加。
- 空气干燥，羽粉问题显现。

### 3.2 养护要点

| 方面 | 建议 |
|---|---|
| 温度 | 逐步过渡，避免骤然降温；夜间注意保温 |
| 湿度 | 干燥时可加湿，折衷等品种需 60%+ |
| 饮食 | 换羽期增加蛋白质（煮熟蛋黄、豆类）与脂肪（少量坚果）；补充维生素与矿物质 |
| 洗澡 | 减少洗澡频次，洗后必须完全干燥（可用毛巾轻擦 + 保温灯）|
| 观察 | 换羽期易患疾病，关注羽毛生长状态 |
| 体检 | 换羽前后是体检与驱虫的好时机 |

### 3.3 换羽期专项

详见第 7 节。秋季换羽最为明显，需重点关注营养与应激。

### 3.4 秋季常见问题

- **羽毛脱落异常**：换羽正常，但若秃斑或啄羽需就医。
- **呼吸道问题**：温差大易感冒，关注温度告警。
- **食欲变化**：食欲增加正常，但需控制高脂食物避免肥胖。

## 4. 冬季养护（12–2 月）

### 4.1 季节特点

- 气温低，需保温。
- 室内供暖导致空气干燥、温差大。
- 鹦鹉活动减少，代谢变化。
- 燃气/煤油取暖存在致命气体风险。

### 4.2 养护要点

| 方面 | 建议 |
|---|---|
| 温度 | 维持 18℃以上，<18℃ 触发 `temp_low` 告警 |
| 保温 | 使用鸟类保温灯或加热垫（垫笼底一侧），配恒温器 |
| 湿度 | 供暖导致干燥，<40% 触发 `humidity_low` 告警，需加湿 |
| 通风 | 中午气温高时短时开窗，避免长时间密闭 |
| 光照 | 冬季日照短，需补光 10–12 小时 |
| 饮食 | 适当增加热量（少量坚果），但避免肥胖 |

### 4.3 冬季禁忌

- ❌ **禁止使用煤油加热器、燃气取暖器**：燃烧产物（一氧化碳、氮氧化物）对鹦鹉致命。
- ❌ 禁止将加热设备直接接触笼子或垫料（火灾、烫伤）。
- ❌ 禁止鹦鹉靠近明火、电热毯。
- ❌ 避免 Canary/鹦鹉所在房间长时间密闭不通风。

### 4.4 温差管理

- 笼舍避免放在暖气旁与门口（温差大）。
- 夜间遮盖笼布保温，留通风口。
- 出门前调好室内温度，避免外出期间温度骤降。

## 5. 梅雨季与回南天

### 5.1 季节特点

- 持续高湿（>80%），部分地区持续数周。
- 物品易发霉，食物易变质。
- 曲霉菌滋生，对鹦鹉呼吸道致命。
- 羽粉聚集，粉尘告警频发。

### 5.2 养护要点

| 方面 | 建议 |
|---|---|
| 除湿 | 每日开除湿机 ≥4 小时，目标湿度 50–60% |
| 通风 | 间歇通风，但外界湿度高时关窗除湿 |
| 食物 | 颗粒粮密封防潮，发现霉变立即丢弃（霉菌毒素致命）|
| 笼具 | 增加清洁频次，垫料每日更换，防止霉菌滋生 |
| 净化 | 开空气净化器（带 HEPA + 除菌） |
| 观察 | 关注打喷嚏、呼吸杂音，疑似曲霉菌感染立即就医 |

### 5.3 曲霉菌预防

曲霉菌（*Aspergillus*）在高湿环境繁殖，吸入孢子可致鹦鹉致命性呼吸道感染（派翁尼斯尤其易感）：

- 保持笼具与垫料干燥。
- 食物防潮，丢弃霉变食物。
- 环境除湿至 60% 以下。
- 出现呼吸症状立即就医，曲霉菌感染治疗难度大。

## 6. 换季期通用注意事项

### 6.1 温差管理

- 换季期单日温差大，是应激与疾病高发期。
- 系统温度告警阈值在换季期可适当放宽（如 `temp_low` 降至 16℃），避免频繁误报。
- 笼舍避免放在温差大的位置（门口、窗边、空调出风口）。

### 6.2 营养过渡

- 换季期逐步调整饮食，避免突然换粮。
- 换羽期增加蛋白质与维生素。
- 冬季适当增加热量，夏季清淡。

### 6.3 应激管理

- 换季期减少环境变化（搬家、换笼、新成员）。
- 保持作息规律，确保 10–12 小时黑暗睡眠。
- 出现行为异常（尖叫、啄羽）及时排查环境因素。

### 6.4 设备检查

- 换季前检查保温灯、加湿器、除湿机、空气净化器是否正常。
- 校准传感器（温度、湿度、粉尘）。
- 备好季节性应急物品。

## 7. 换羽期专项养护

### 7.1 换羽规律

- 鹦鹉通常每年换羽 1–2 次，春秋季为高峰。
- 换羽持续数周到数月，新羽毛生长需要大量营养。
- 换羽期鹦鹉可能情绪低落、易怒、羽毛凌乱（正常现象）。

### 7.2 营养支持

| 营养素 | 食物来源 | 作用 |
|---|---|---|
| 蛋白质 | 煮熟蛋黄、豆类、少量煮熟鸡肉 | 羽毛角蛋白合成 |
| 脂肪 | 少量坚果（核桃、杏仁）、亚麻籽 | 羽毛光泽 |
| 钙 | 墨鱼骨、矿物质块 | 羽毛结构、血羽生长 |
| 维生素 A | 深绿蔬菜、胡萝卜、南瓜 | 皮肤与羽毛健康 |
| 维生素 E | 少量麦胚、坚果 | 抗氧化、羽毛生长 |

### 7.3 换羽期护理

- 增加洗澡频次，软化旧羽鞘，便于脱落。
- 提供啃咬玩具，缓解换羽期情绪。
- 避免抓握鹦鹉（血羽易折断出血）。
- 关注血羽生长，出血时按 [鹦鹉急救指南.md](鹦鹉急救指南.md) 第 6 节处理。

### 7.4 异常换羽

- 秃斑、长期不换羽、啄羽：需就医排查 PBFD、营养缺乏、寄生虫。
- 换羽期体重明显下降：需就医。

## 8. 繁殖期专项养护

### 8.1 繁殖期表现

- 雄性：攻击性增强、求偶舞蹈、叫声增多。
- 雌性：寻找巢穴、产蛋（即使无雄性也会产未受精蛋）、腹部膨大。

### 8.2 雌鸟养护

- 提供充足钙质（产蛋消耗大）。
- 关注蛋难产风险（参见 [鹦鹉急救指南.md](鹦鹉急救指南.md) 第 10 节）。
- 不打算繁殖的雌鸟：减少筑巢刺激（移除巢箱、减少暗处栖息），控制光照 12 小时以内。

### 8.3 行为管理

- 繁殖期攻击性增强，避免强行互动。
- 用栖木引导而非手抓。
- 啄羽症在繁殖期高发，关注情绪与环境。

## 9. 系统季节性配置建议

根据季节调整系统阈值与联动，提升告警准确性：

### 9.1 阈值调整

| 季节 | 温度告警 | 湿度告警 | 粉尘告警 |
|---|---|---|---|
| 春季 | low=16, high=30 | low=35, high=70 | 默认 |
| 夏季 | low=18, high=30 | low=40, high=70（梅雨 low=40, high=75）| 默认 |
| 秋季 | low=16, high=30 | low=35, high=70 | 默认 |
| 冬季 | low=18, high=28 | low=40, high=70 | 默认 |
| 梅雨季 | 默认 | low=40, high=65（严格）| warning 下调至 25 |

> 修改路径：系统设置 → 阈值配置。修改后需确认缓存刷新。

### 9.2 联动设备调整

| 季节 | 排风扇 | 空气净化器 | 加湿器 | 除湿机 |
|---|---|---|---|---|
| 春季 | 常开 | 常开 | 按需 | 按需 |
| 夏季 | 常开 | 常开 | 关闭 | 梅雨季常开 |
| 秋季 | 按需 | 常开 | 按需 | 按需 |
| 冬季 | 按需 | 常开 | 常开 | 关闭 |
| 梅雨季 | 间歇 | 常开 | 关闭 | 常开 |

### 9.3 季节性提醒

- 系统可在成长报告中按季节推送养护建议（如「秋季换羽期，建议增加蛋白质」）。
- 换季前一周推送设备检查提醒。
