## 1. 基础概述 / Overview

### 1.1 鹦鹉的定义 / What Is a Parrot

鹦鹉是鹦形目（*Psittaciformes*）鸟类的统称，目前已知约有 398 个物种，隶属于 92 个属。它们是最具认知能力的鸟类类群之一，以其弯曲有力的钩状喙、对趾型足（两趾向前、两趾向后）、鲜艳的羽色以及模仿人类语言的能力而闻名。

Parrots are birds of the order *Psittaciformes*, comprising approximately 398 species across 92 genera. They are among the most cognitively advanced bird groups, recognized by their strong curved hooked beaks, zygodactyl feet (two toes forward, two backward), vibrant plumage, and ability to mimic human speech.

### 1.2 分类学地位 / Taxonomic Position

- **界 / Kingdom**: 动物界 / Animalia
- **门 / Phylum**: 脊索动物门 / Chordata
- **纲 / Class**: 鸟纲 / Aves
- **目 / Order**: 鹦形目 / *Psittaciformes*

鹦形目下主要分为三个总科 / The order is divided into three main superfamilies:

- **鹦鹉总科 / Superfamily *Psittacoidea***（真鹦鹉 / True parrots）：包括金刚鹦鹉、亚马逊鹦鹉、非洲灰鹦鹉、虎皮鹦鹉等绝大多数常见宠物鹦鹉
- **凤头鹦鹉总科 / Superfamily *Cacatuoidea***（凤头鹦鹉 / Cockatoos）：包括葵花凤头鹦鹉、戈芬氏凤头鹦鹉等，特征为可竖立的冠羽
- **新西兰鹦鹉总科 / Superfamily *Strigopoidea***：包括鸮鹦鹉（Kakapo）、啄羊鹦鹉（Kea）等，仅分布于新西兰及周边

### 1.3 地理分布 / Geographic Distribution

鹦鹉主要分布于全球热带和亚热带地区，集中分布在以下区域：

- **南美洲 / South America**：物种多样性最高，尤其是亚马逊雨林地区，是金刚鹦鹉、亚马逊鹦鹉的原产地
- **大洋洲 / Oceania**：澳大利亚是鹦鹉最丰富的国家之一，虎皮鹦鹉、鸡尾鹦鹉、吸蜜鹦鹉等原产于此
- **非洲 / Africa**：非洲灰鹦鹉、牡丹鹦鹉等原产于非洲大陆及马达加斯加
- **亚洲南部 / South & Southeast Asia**：环颈鹦鹉、月轮鹦鹉、鹦鹉公主等原产于印度、东南亚地区
- **中美洲 / Central America**：部分金刚鹦鹉和亚马逊鹦鹉分布至此

鹦鹉不自然分布于北极、南极及大部分温带地区，但已被引入全球多地作为宠物饲养。

### 1.4 体型特征 / Physical Characteristics

鹦鹉体型差异极大，从最小的侏鹦鹉到最大的金刚鹦鹉，跨度显著：

- **最小物种 / Smallest species**: 侏鹦鹉属（*Micropsitta*），体长约 8–10 cm，体重约 10–15 g
- **最大物种 / Largest species**: 蓝紫金刚鹦鹉（*Anodorhynchus hyacinthinus*），体长可达 100 cm（含尾羽），体重约 1.4–1.7 kg
- **常见宠物鹦鹉体长 / Common pet parrot length**: 12–50 cm

鹦鹉的共同特征包括：
- 钩状喙 / Hooked beak：上喙弯曲，可咬碎坚果和种子
- 对趾型足 / Zygodactyl feet：两趾向前两趾向后，适合攀爬和抓握食物
- 厚舌 / Thick muscular tongue：可灵活操控食物，部分物种舌头尖端有触觉
- 羽毛色素 / Plumage pigments：含特殊的鹦鹉色素（psittacofulvins），产生鲜艳的红、黄、橙色

### 1.5 寿命范围 / Lifespan

鹦鹉是鸟类中最长寿的类群之一，寿命与体型正相关：

| 体型 / Size | 代表物种 / Example Species | 平均寿命 / Average Lifespan |
|---|---|---|
| 小型 / Small | 虎皮鹦鹉 / Budgerigar | 5–10 年 |
| 中型 / Medium | 玄凤鹦鹉 / Cockatiel | 15–25 年 |
| 中大型 / Medium-Large | 非洲灰鹦鹉 / African Grey | 40–60 年 |
| 大型 / Large | 金刚鹦鹉 / Macaw | 50–80 年 |

在良好饲养条件下，许多鹦鹉可超过其平均寿命。葵花凤头鹦鹉和金刚鹦鹉有记录活到 100 岁以上的案例。

## 2. 常见饲养品种 / Popular Pet Species

### 2.1 虎皮鹦鹉 / Budgerigar (*Melopsittacus undulatus*)

- **原产地 / Origin**: 澳大利亚内陆地区
- **体长 / Length**: 约 18 cm（含尾羽）
- **体重 / Weight**: 30–40 g
- **寿命 / Lifespan**: 5–10 年（良好饲养可达 15 年）
- **学话能力 / Talking ability**: ★★★★☆（雄性可学会大量词汇，有记录掌握超过 1700 个词）
- **性格特点 / Temperament**: 活泼好动、好奇心强、群居性极强
- **适养人群 / Suitable for**: 初次养鸟者、家庭、儿童（需监督互动）
- **特殊喜好 / Special preferences**: 喜欢洗澡、喜欢镜子（但长期照镜可能导致行为问题）、喜欢探索新玩具
- **注意事项 / Notes**: 需成对或成群饲养以避免孤独；羽色变异极多（蓝、黄、绿、白、灰等）

### 2.2 玄凤鹦鹉 / Cockatiel (*Nymphicus hollandicus*)

- **别名 / Aliases**: 鸡尾鹦鹉、玄凤
- **原产地 / Origin**: 澳大利亚
- **体长 / Length**: 约 30–33 cm
- **体重 / Weight**: 80–120 g
- **寿命 / Lifespan**: 15–25 年
- **学话能力 / Talking ability**: ★★★☆☆（雄性可学简单词汇和口哨，不如虎皮鹦鹉擅长学话，但擅长吹口哨）
- **性格特点 / Temperament**: 温和亲人、喜欢被抚摸、冠羽表达情绪（竖起=警觉/兴奋，平放=放松，向后=恐惧）
- **适养人群 / Suitable for**: 初次至中级养鸟者、家庭、独居者
- **特殊喜好 / Special preferences**: 喜欢被挠头部和脸颊、喜欢口哨互动、喜欢在主人肩膀上停留
- **注意事项 / Notes**: 容易患脂肪瘤和呼吸系统疾病；对空气质量敏感；雌性可能面临产蛋相关健康问题（如蛋难产）

### 2.3 非洲灰鹦鹉 / African Grey Parrot (*Psittacus erithacus*)

- **亚种 / Subspecies**: 刚果非洲灰（*P. e. erithacus*，体型较大）和提姆那灰（*P. e. timneh*，体型较小）
- **原产地 / Origin**: 非洲中西部赤道森林
- **体长 / Length**: 约 33 cm
- **体重 / Weight**: 400–650 g
- **寿命 / Lifespan**: 40–60 年
- **学话能力 / Talking ability**: ★★★★★（被公认为最聪明的鹦鹉，能理解词义、使用词汇进行有意义的交流）
- **性格特点 / Temperament**: 极度聪明、敏感、情感丰富、需要大量精神刺激
- **适养人群 / Suitable for**: 有经验的养鸟者、能投入大量时间互动的人
- **特殊喜好 / Special preferences**: 喜欢解谜玩具、喜欢学习新词汇、喜欢与主人深度互动
- **注意事项 / Notes**: 容易因无聊或压力产生啄羽症；对生活环境变化敏感；需要丰富的环境丰容；著名实验鹦鹉"Alex"（由 Irene Pepperberg 博士训练）展示了相当于 5 岁儿童的认知水平

### 2.4 蓝黄金刚鹦鹉 / Blue-and-Gold Macaw (*Ara ararauna*)

- **别名 / Aliases**: 蓝黄金刚、琉璃金刚
- **原产地 / Origin**: 南美洲（委内瑞拉、巴西、玻利维亚等）
- **体长 / Length**: 约 76–86 cm
- **体重 / Weight**: 900–1200 g
- **寿命 / Lifespan**: 50–80 年
- **学话能力 / Talking ability**: ★★★☆☆（可学会一些词汇，但不如非洲灰鹦鹉）
- **性格特点 / Temperament**: 外向活泼、友善亲人、声音洪亮（可达 100 分贝以上）
- **适养人群 / Suitable for**: 有经验的养鸟者、有足够空间和时间的饲养者
- **特殊喜好 / Special preferences**: 喜欢与主人互动、喜欢啃咬木头、喜欢洗澡
- **注意事项 / Notes**: 叫声极大，不适合公寓饲养；需要大型飞行笼；啃咬力极强，可破坏家具；需要大量社交互动

### 2.5 绿翅金刚鹦鹉 / Green-Winged Macaw (*Ara chloropterus*)

- **原产地 / Origin**: 南美洲（哥伦比亚、委内瑞拉至阿根廷北部）
- **体长 / Length**: 约 81–90 cm
- **体重 / Weight**: 1000–1700 g
- **寿命 / Lifespan**: 50–80 年
- **学话能力 / Talking ability**: ★★★☆☆
- **性格特点 / Temperament**: 温和亲人、比蓝黄金刚更安静、深情
- **适养人群 / Suitable for**: 有经验的养鸟者
- **特殊喜好 / Special preferences**: 喜欢与主人亲密接触、喜欢被拥抱
- **注意事项 / Notes**: 胸部和头部有红色羽毛，翅膀有蓝绿色羽毛；容易产生分离焦虑

### 2.6 葵花凤头鹦鹉 / Sulphur-Crested Cockatoo (*Cacatua galerita*)

- **原产地 / Origin**: 澳大利亚东部和北部、新几内亚
- **体长 / Length**: 约 44–51 cm
- **体重 / Weight**: 800–950 g
- **寿命 / Lifespan**: 40–80 年
- **学话能力 / Talking ability**: ★★☆☆☆（学话能力一般，但擅长模仿声音）
- **性格特点 / Temperament**: 极度亲人、需要大量关注、容易产生行为问题
- **适养人群 / Suitable for**: 有丰富经验的养鸟者、能全天陪伴的饲养者
- **特殊喜好 / Special preferences**: 喜欢跳舞、喜欢竖起冠羽表达兴奋、喜欢与主人互动
- **注意事项 / Notes**: 叫声极大；容易因无聊产生啄羽症和自残行为；需要大量精神刺激；是全球宠物救助中最常见的被遗弃大型鹦鹉之一

### 2.7 戈芬氏凤头鹦鹉 / Goffin's Cockatoo (*Cacatua goffiniana*)

- **别名 / Aliases**: 戈芬氏葵花
- **原产地 / Origin**: 印度尼西亚塔劳群岛
- **体长 / Length**: 约 32 cm
- **体重 / Weight**: 约 300 g
- **寿命 / Lifespan**: 30–40 年
- **学话能力 / Talking ability**: ★★☆☆☆
- **性格特点 / Temperament**: 好奇、爱玩、聪明、精力充沛
- **适养人群 / Suitable for**: 有经验的养鸟者
- **特殊喜好 / Special preferences**: 喜欢拆解玩具、喜欢探索
- **注意事项 / Notes**: 体型较小但需求不减；研究显示具有工具使用能力

### 2.8 亚马逊鹦鹉 / Amazon Parrot

常见宠物亚马逊品种包括：

- **黄头亚马逊 / Yellow-Headed Amazon (*Amazona oratrix*)**
  - 体长 / Length: 约 38–43 cm
  - 寿命 / Lifespan: 50–80 年
  - 学话能力 / Talking ability: ★★★★★（最佳学话品种之一）
  - 性格 / Temperament: 外向、爱唱歌、有领地意识（繁殖期可能攻击性增强）

- **蓝顶亚马逊 / Blue-Fronted Amazon (*Amazona aestiva*)**
  - 体长 / Length: 约 36–38 cm
  - 寿命 / Lifespan: 40–80 年
  - 学话能力 / Talking ability: ★★★★★
  - 性格 / Temperament: 聪明、活泼、喜欢互动

- **黄颈亚马逊 / Yellow-Naped Amazon (*Amazona auropalliata*)**
  - 体长 / Length: 约 38 cm
  - 寿命 / Lifespan: 50–80 年
  - 学话能力 / Talking ability: ★★★★★（被公认为学话最清晰的亚马逊）
  - 性格 / Temperament: 深情、忠诚、可能只与一人建立深厚关系

### 2.9 环颈鹦鹉 / Ringneck Parakeet (*Psittacula krameri*)

- **别名 / Aliases**: 红领绿鹦鹉、月轮鹦鹉（月轮为另一近似种 *Psittacula alexandri*）
- **原产地 / Origin**: 非洲和亚洲（印度、东南亚）
- **体长 / Length**: 约 40 cm（含长尾羽）
- **体重 / Weight**: 约 110–140 g
- **寿命 / Lifespan**: 25–30 年
- **学话能力 / Talking ability**: ★★★★☆（雄性学话能力尤佳）
- **性格特点 / Temperament**: 独立、聪明、有个性
- **适养人群 / Suitable for**: 中级至高级养鸟者
- **特殊喜好 / Special preferences**: 喜欢攀爬、喜欢学习技巧
- **注意事项 / Notes**: 有"青春期"行为变化（约 2–4 岁期间可能变得有攻击性）；雄性颈部有明显环纹，雌性无

### 2.10 牡丹鹦鹉 / Lovebird (*Agapornis* spp.)

常见品种包括：
- 桃面牡丹 / Peach-Faced Lovebird (*Agapornis roseicollis*)
- 费氏牡丹 / Fischer's Lovebird (*Agapornis fischeri*)
- 黑脸牡丹 / Masked Lovebird (*Agapornis personatus*)

- **原产地 / Origin**: 非洲和马达加斯加
- **体长 / Length**: 约 13–17 cm
- **体重 / Weight**: 40–60 g
- **寿命 / Lifespan**: 10–20 年

- **特殊喜好 / Special preferences**: 喜欢在垫料中挖掘、喜欢成对活动
- **注意事项 / Notes**: 名称来源于伴侣间的亲密行为；单独饲养需更多主人互动；可能对其他鸟类有攻击性

### 2.11 吸蜜鹦鹉 / Lory / Lorikeet

- 黑顶吸蜜 / Black-Capped Lorikeet (*Lorius lory*)

- **原产地 / Origin**: 澳大利亚、新几内亚、印度尼西亚东部
- **体长 / Length**: 约 25–30 cm
- **体重 / Weight**: 100–180 g
- **寿命 / Lifespan**: 15–25 年
- **学话能力 / Talking ability**: ★★★☆☆
- **性格特点 / Temperament**: 活泼、好奇、爱玩
- **适养人群 / Suitable for**: 中级养鸟者
- **特殊喜好 / Special preferences**: 喜欢花蜜和软质食物、喜欢洗澡（会全身湿透）
- **注意事项 / Notes**: 饮食特殊（以花蜜、花粉、软果为主），粪便稀湿，清洁频率需更高；不适合对卫生要求苛刻的家庭

### 2.12 其他常见品种 / Other Common Species

  - 澳大利亚原产，羽色柔和（粉棕、灰、蓝），性格极温和安静
  - 适合喜欢观察但不需要频繁互动的饲养者
  - 体长 / Length: 约 19–23 cm；寿命 / Lifespan: 10–15 年

## 3. 饲养与护理 / Care & Husbandry

### 3.1 笼具选择 / Cage Selection

**基本原则 / Basic Principles**:
- 笼子宽度应至少能让鹦鹉完全展开双翼而不触碰笼壁
- 笼子应宽大于高（鹦鹉水平飞行而非垂直）
- 笼条间距需适合鹦鹉体型（小型鹦鹉 ≤ 1.5 cm，大型鹦鹉 ≤ 2.5–3 cm）

**推荐尺寸 / Recommended Sizes**:

| 鹦鹉类型 / Parrot Type | 最小笼宽×深×高 / Min Cage W×D×H |
|---|---|
| 虎皮/牡丹 / Budgie/Lovebird | 45×45×60 cm |
| 玄凤/吸蜜 / Cockatiel/Lorikeet | 60×45×60 cm |
| 环颈/派翁尼斯 / Ringneck/Pionus | 75×60×90 cm |
| 亚马逊/灰鹦鹉 / Amazon/Grey | 90×60×120 cm |
| 金刚/凤头 / Macaw/Cockatoo | 120×90×150 cm 以上 |

**材质 / Materials**:
- 首选不锈钢或粉末涂层钢材（无毒、耐用、易清洁）
- 避免镀锌笼（锌中毒风险）
- 避免铁丝笼涂漆剥落（铅中毒风险）

**笼内配件 / Cage Accessories**:
- 栖木：不同粗细的天然木枝（如苹果木、柳木），直径 1.5–3 cm 不等
- 食盆和水盆：不锈钢或陶瓷材质，每日清洁
- 玩具：啃咬玩具、解谜玩具、秋千、梯子等
- 墨鱼骨或矿物质块：补充钙质和矿物质

### 3.2 饮食营养 / Diet & Nutrition

**基础饮食组成 / Basic Diet Composition**:

| 食物类别 / Food Category | 占比 / Percentage | 说明 / Notes |
|---|---|---|
| 颗粒饲料 / Pelleted food | 50–70% | 全营养配方，避免挑食 |
| 新鲜蔬菜 / Fresh vegetables | 20–30% | 深绿色叶菜、胡萝卜、西兰花等 |
| 新鲜水果 / Fresh fruits | 5–10% | 苹果、葡萄、浆果等（限量） |
| 种子和坚果 / Seeds & nuts | 5–10% | 作为零食或训练奖励 |

**推荐蔬菜 / Recommended Vegetables**:
- 深绿色叶菜：羽衣甘蓝、菠菜（限量）、芥菜、西兰花
- 橙色蔬菜：胡萝卜、南瓜、红薯（煮熟）
- 其他：甜椒、黄瓜、西芦笋、玉米

**推荐水果 / Recommended Fruits**:
- 苹果（去籽）、梨（去籽）、葡萄、蓝莓、草莓、香蕉、芒果、木瓜
- 柑橘类少量（可能引起消化不适）

**有毒食物清单 / TOXIC FOODS — STRICTLY AVOID**:

| 有毒食物 / Toxic Food | 毒性成分 / Toxic Compound | 中毒症状 / Symptoms |
|---|---|---|
| 牛油果 / Avocado |  Persin（果素） | 呼吸困难、心力衰竭、猝死 |
| 巧克力 / Chocolate | 可可碱 / Theobromine | 心律不齐、抽搐、死亡 |
| 咖啡因 / Caffeine | 咖啡因 | 心律不齐、过度兴奋、死亡 |
| 洋葱和大蒜 / Onion & Garlic | 硫代硫酸盐 | 溶血性贫血 |
| 苹果/梨籽 / Apple/Pear seeds | 氰苷（释放氰化物） | 氰化物中毒 |
| 酒精 / Alcohol | 乙醇 | 中枢神经抑制、死亡 |
| 高盐食物 / High-salt foods | 钠 | 钠中毒、肾损伤 |
| 高脂肪食物 / High-fat foods | 脂肪 | 脂肪肝、肥胖 |
| 生豆类 / Raw beans | 植物凝集素 | 消化系统损伤 |
| 蘑菇 / Mushrooms | 多种毒素 | 肝肾损伤 |
| 番茄叶和茎 / Tomato leaves & stems | 番茄碱 | 消化不适 |
| 木糖醇 / Xylitol | 木糖醇 | 低血糖、肝衰竭 |

**补充剂 / Supplements**:
- 维生素 A：鹦鹉易缺乏，可通过深绿色和橙色蔬菜补充
- 钙质：墨鱼骨、矿物质块、钙粉（尤其产蛋期雌鸟）
- 避免过量维生素补充（脂溶性维生素 A、D、E、K 过量可中毒）

### 3.3 环境要求 / Environmental Requirements

**温度 / Temperature**:
- 适宜范围：18–26°C（65–79°F）
- 避免温度骤变（>5°C 的突然变化可导致应激）
- 避免冷风直吹和空调直吹
- 冬季可适当加温，但避免使用煤油加热器（释放有毒气体）

**湿度 / Humidity**:
- 适宜范围：40–60%
- 干燥环境可导致皮肤和呼吸问题
- 可使用加湿器或定期喷雾洗澡增加湿度

**光照 / Lighting**:
- 需要自然光或全光谱灯光（含 UVB）
- 每日 10–12 小时光照，12–14 小时黑暗（模拟自然昼夜）
- UVB 有助于维生素 D3 合成和钙质吸收
- 避免将笼子放在窗边直射（温度骤变和天敌惊吓风险）

**通风 / Ventilation**:
- 保持空气流通但避免穿堂风
- 避免厨房油烟（特氟龙/PTFE 涂层锅加热释放的气体对鹦鹉致命）
- 避免使用空气清新剂、香薰蜡烛、香烟

### 3.4 日常护理 / Daily Care

**洗澡 / Bathing**:
- 大多数鹦鹉喜欢洗澡，频率因品种而异
- 提供浅水盆或喷雾瓶轻柔喷淋
- 吸蜜鹦鹉尤其喜欢全身湿透的洗澡方式
- 洗澡后避免着凉，确保羽毛完全干燥后再入睡

**剪羽 / Wing Trimming**:
- 可选择性剪羽以限制飞行能力（安全考虑）
- 仅修剪初级飞羽，保留覆盖羽
- 应由有经验者操作，避免剪到血羽（有血管的活羽）
- 剪羽是暂时性的，换羽后需重新修剪
- 争议：部分饲养者主张不剪羽以保持鹦鹉自然行为

**指甲修剪 / Nail Trimming**:
- 定期检查指甲长度，过长可能卡入栖木或抓伤主人
- 使用专用鸟用指甲剪，避免剪到血线（指甲内的血管）
- 可提供不同粗细的栖木帮助自然磨损

**喙部护理 / Beak Care**:
- 健康鹦鹉的喙通常无需修剪
- 提供墨鱼骨、矿物质块、天然木枝帮助磨喙
- 喙部异常生长可能提示肝脏疾病或其他健康问题

### 3.5 健康监测 / Health Monitoring

**健康鸟的体征 / Signs of a Healthy Parrot**:
- 眼睛明亮、有神
- 羽毛光滑、紧贴身体
- 食欲良好、进食活跃
- 粪便成形（由深绿色固体部分、白色尿酸部分组成）
- 行为活跃、警觉
- 呼吸平稳、无杂音

**疾病警示信号 / Warning Signs of Illness**:
- 精神萎靡、嗜睡
- 食欲下降或拒食
- 羽毛蓬松、长时间"炸毛"
- 呼吸困难、张口呼吸、尾羽随呼吸摆动
- 粪便颜色或形态异常（持续稀便、黑色、红色）
- 呕吐或反流
- 体重突然下降
- 眼部或鼻腔分泌物
- 行为异常（过度尖叫、攻击性、自残）

**常见疾病 / Common Diseases**:

| 疾病 / Disease | 症状 / Symptoms | 原因 / Cause |
|---|---|---|
| PBFD（喙羽病）/ Psittacine Beak and Feather Disease | 羽毛异常生长、脱落、喙变形 | 圆环病毒 / Circovirus |
| 鹦鹉热 / Psittacosis（衣原体病） | 呼吸症状、绿色稀便、结膜炎 | 鹦鹉衣原体 / *Chlamydia psittaci* |
| 啄羽症 / Feather Plucking | 自行啄掉羽毛 | 行为问题、寄生虫、过敏、疾病 |
| 脂肪肝 / Fatty Liver Disease | 肥胖、喙部过度生长、腹泻 | 高脂饮食（尤其种子为主） |
| 蛋难产 / Egg Binding | 腹部肿胀、蹲伏、精神差 | 钙质缺乏、过度产蛋 |
| 呼吸系统感染 / Respiratory Infection | 打喷嚏、呼吸杂音、张口呼吸 | 细菌、真菌、环境因素 |
| 重金属中毒 / Heavy Metal Poisoning | 神经症状、呕吐、绿色粪便 | 摄入铅、锌等金属 |
| 痛风 / Gout | 关节肿胀、疼痛 | 肾脏疾病、高蛋白饮食 |

**兽医护理 / Veterinary Care**:
- 建议每年进行一次鸟类专科兽医体检
- 新鸟引进需隔离观察 30–45 天
- 保存鸟类兽医的联系方式以备急诊
- 鹦鹉隐藏疾病的能力很强（猎物动物本能），出现症状时往往已较严重

### 3.6 家居安全 / Household Safety

**危险源 / Hazards**:

| 危险源 / Hazard | 风险 / Risk |
|---|---|
| 特氟龙/PTFE 涂层锅 / Teflon/PTFE cookware | 过热释放气体，可在数分钟内致命 |
| 不粘锅 / Non-stick cookware | 同上 |
| 铅基油漆 / Lead-based paint | 重金属中毒 |
| 镀锌笼具 / Galvanized cages | 锌中毒 |
| 开放式水源（马桶、水桶）/ Open water sources | 溺水 |
| 风扇运转中 / Ceiling fans | 飞行中碰撞受伤 |
| 镜子、玻璃 / Mirrors, glass | 飞行碰撞 |
| 电线 / Electrical cords | 啃咬触电 |
| 有毒植物 / Toxic plants | 摄入中毒（如杜鹃花、常春藤、万年青等） |
| 其他宠物（猫狗）/ Other pets | 抓咬伤害（即使看似友好） |
| 香烟烟雾 / Cigarette smoke | 呼吸系统疾病 |
| 空气清新剂、香薰 / Air fresheners, candles | 呼吸系统刺激 |

## 4. 行为与训练 / Behavior & Training

### 4.1 社交行为 / Social Behavior

鹦鹉是高度社会化的群居动物，在野外通常以群体形式生活。这一特性决定了宠物鹦鹉需要大量的社交互动：

- **群居本能 / Flock Instinct**: 宠物鹦鹉将人类家庭成员视为自己的"鸟群"，需要与"鸟群"成员互动
- **配对关系 / Pair Bonding**: 鹦鹉可能与特定一人建立深厚的配对关系，对其他人表现出攻击性
- **日常社交需求 / Daily Social Needs**: 大多数宠物鹦鹉每天需要 2–4 小时的直接互动时间
- **孤独的影响 / Effects of Isolation**: 长期孤独可导致抑郁、啄羽症、过度尖叫等行为问题

**建议 / Recommendations**:
- 如果主人白天不在家，考虑饲养两只鹦鹉作伴
- 将鸟笼放在家庭活动频繁的区域（如客厅），让鹦鹉观察家庭活动
- 与鹦鹉进行日常互动：说话、训练、抚摸、一起活动

### 4.2 学话训练 / Speech Training

**最佳时机 / Best Time to Start**:
- 最佳开始年龄：6 个月至 2 岁（年轻鹦鹉学习能力最强）
- 非洲灰鹦鹉和亚马逊鹦鹉可在任何年龄学习新词汇
- 雄性虎皮鹦鹉和玄凤鹦鹉学话能力优于雌性

**训练方法 / Training Methods**:

1. **重复法 / Repetition Method**
   - 每天在固定时间重复简单词汇或短语
   - 使用兴奋的语调，配合实物或动作
   - 例如：拿着苹果说"苹果 / apple"，给鸟吃时再说一遍

2. **关联法 / Association Method**
   - 将词汇与特定情境关联
   - 进门时说"你好 / hello"，出门时说"再见 / bye-bye"
   - 喂食前说"好吃 / yummy"

3. **模仿法 / Mimicry Method**
   - 鹦鹉天生喜欢模仿声音
   - 播放录音或让鹦鹉听其他会说话的鸟
   - 注意：鹦鹉也会模仿不希望出现的声音（如尖叫、门铃）

4. **正向强化 / Positive Reinforcement**
   - 当鹦鹉尝试发声或说出词汇时，立即给予食物奖励和口头表扬
   - 不要惩罚"错误"的尝试
   - 保持训练时间短（5–10 分钟），频率高（每日多次）

**学话能力排名 / Talking Ability Ranking**:

| 排名 / Rank | 品种 / Species | 词汇量潜力 / Vocabulary Potential | 清晰度 / Clarity |
|---|---|---|---|
| 1 | 非洲灰鹦鹉 / African Grey | 1000+ 词 | 极高 |
| 2 | 黄颈亚马逊 / Yellow-Naped Amazon | 500+ 词 | 极高 |
| 3 | 蓝顶亚马逊 / Blue-Fronted Amazon | 300+ 词 | 高 |
| 4 | 虎皮鹦鹉（雄性）/ Budgie (male) | 100–1000+ 词 | 中等（有记录达 1723 词） |
| 5 | 环颈鹦鹉 / Ringneck | 200+ 词 | 高 |
| 6 | 凤头鹦鹉 / Cockatoo | 50–100 词 | 中等 |
| 7 | 金刚鹦鹉 / Macaw | 50–100 词 | 中等 |
| 8 | 玄凤鹦鹉 / Cockatiel | 20–50 词 | 低（擅长口哨） |
| 9 | 牡丹鹦鹉 / Lovebird | 极少 | 极低 |

### 4.3 常见问题行为及纠正 / Common Behavior Problems

**过度尖叫 / Excessive Screaming**:

- **原因 / Causes**: 寻求关注、无聊、焦虑、习惯性
- **纠正方法 / Solutions**:
  - 不要对尖叫给予任何回应（包括责骂，因为任何回应都是强化）
  - 在安静时给予关注和奖励
  - 增加环境丰容和互动时间
  - 建立规律的作息时间
  - 使用"安静"指令训练

**咬人 / Biting**:

- **原因 / Causes**: 恐惧、领地意识、保护巢穴、玩耍、激素变化
- **纠正方法 / Solutions**:
  - 不要猛抽手（会伤害鹦鹉喙部），保持稳定
  - 不要惩罚咬人（会加剧恐惧）
  - 观察咬人的前兆信号（瞳孔收缩、身体紧张、喙部微张）
  - 用"上来"指令转移注意力
  - 提供啃咬玩具满足咬的需求
  - 理解咬人是鹦鹉的沟通方式，找出原因

**啄羽症 / Feather Plucking**:

- **原因 / Causes**:
  - 行为性：无聊、压力、焦虑、缺乏社交
  - 医学性：寄生虫、过敏、感染、内分泌疾病、营养缺乏
- **纠正方法 / Solutions**:
  - 首先排除医学原因（就医检查）
  - 增加环境丰容和互动时间
  - 提供解谜玩具和觅食玩具
  - 确保充足睡眠（10–12 小时黑暗）
  - 使用费洛蒙喷雾（如鸟用镇定喷雾）
  - 严重时可能需要使用颈圈（伊丽莎白圈）阻止啄羽

**攻击性 / Aggression**:

- **原因 / Causes**: 领地意识、恐惧、激素变化（繁殖期）、保护巢穴
- **纠正方法 / Solutions**:
  - 避免在繁殖期过度刺激鹦鹉
  - 不要用手进入鹦鹉的笼内领地
  - 使用栖木引导而非手抓
  - 建立信任关系
  - 理解攻击性背后的原因

### 4.4 环境丰容 / Environmental Enrichment

环境丰容是保持鹦鹉心理健康的关键：

**觅食丰容 / Foraging Enrichment**:
- 将食物藏在玩具中，让鹦鹉"工作"获取食物
- 使用觅食盒、觅食球、纸袋包裹食物
- 模拟野外觅食行为（鹦鹉在野外花费 60–80% 的时间觅食）

**啃咬丰容 / Chewing Enrichment**:
- 提供天然木枝、皮革条、纸板、棕榈叶
- 金刚鹦鹉和凤头鹦鹉尤其需要大量啃咬
- 定期更换啃咬玩具

**智力丰容 / Intellectual Enrichment**:
- 解谜玩具（滑块、锁扣、齿轮）
- 训练新技巧（转圈、鞠躬、取物）
- 播放音乐或自然声音

**感官丰容 / Sensory Enrichment**:
- 不同材质的玩具（木头、绳子、皮革、塑料）
- 不同颜色的玩具
- 安全的天然草本植物

### 4.5 建立信任 / Building Trust

**初期适应期 / Initial Adjustment Period**:
- 新鸟到家后给予 3–7 天适应期，减少打扰
- 将鸟笼放在安静但能看到家庭活动的位置
- 说话轻柔、动作缓慢

**信任建立步骤 / Trust-Building Steps**:

1. **阶段 1 / Phase 1**: 让鹦鹉习惯你的存在
   - 坐在鸟笼旁读书或工作，不直视鹦鹉
   - 轻声说话或唱歌

2. **阶段 2 / Phase 2**: 让鹦鹉接受你的手
   - 在笼内放置食物，让鹦鹉在你手附近进食
   - 逐渐缩短手与食物的距离

3. **阶段 3 / Phase 3**: 训练"上来"指令
   - 用栖木或手指轻触鹦鹉腹部
   - 说"上来 / step up"，给予食物奖励
   - 逐渐延长停留时间

4. **阶段 4 / Phase 4**: 互动和抚摸
   - 大多数鹦鹉喜欢被挠头部和脸颊（这些部位鹦鹉自己无法梳理）
   - 避免抚摸背部或翅膀（可能被误解为交配行为）

**重要原则 / Important Principles**:
- 永远不要惩罚鹦鹉（会破坏信任）
- 保持一致性（相同的指令、相同的奖励）
- 尊重鹦鹉的"不"（如果鹦鹉不想互动，不要强迫）
- 训练时间宜短不宜长（5–15 分钟）

## 5. 习性、喜好与趣闻 / Habits, Preferences & Fun Facts

### 5.1 左右脚偏好 / Foot Preference（"利手性"）

鹦鹉是少数表现出明显"利手性"（foot laterality）的动物之一：

- 约 70–80% 的鹦鹉偏好使用左爪抓取食物
- 左爪偏好的鹦鹉通常右脑更活跃
- 利手性与学话能力有关联：左爪偏好的鹦鹉词汇量通常更大
- 幼年鹦鹉的利手性不明显，随年龄增长逐渐显现
- 不同品种的利手性比例有所不同

### 5.2 音乐与节奏感 / Music & Rhythm

鹦鹉是除人类外已知唯一能随音乐"跳舞"（即对节奏产生同步运动反应）的动物：

- 著名案例：葵花凤头鹦鹉"Snowball"因随后街男孩歌曲跳舞的视频而广为人知
- 研究（由 Aniruddh Patel 和 John Iversen 等学者进行）证实鹦鹉具有真正的节拍同步能力
- 鹦鹉对节奏的反应是后天学习而非天生本能
- 不同鹦鹉对音乐类型的偏好可能不同
- 播放音乐可作为环境丰容的一部分

### 5.3 认知能力 / Cognitive Abilities

鹦鹉的认知能力可与灵长类动物媲美：

- **非洲灰鹦鹉"Alex"**: 由 Irene Pepperberg 博士训练，能识别 50 种物体、7 种颜色、5 种形状，理解"相同/不同"概念，能数到 6，认知水平相当于 5 岁儿童
- **工具使用**: 戈芬氏凤头鹦鹉和新喀里多尼亚乌鸦（非鹦鹉）能制造和使用工具
- **问题解决**: 许多鹦鹉能解决多步骤的解谜任务
- **情感能力**: 鹦鹉能体验快乐、恐惧、愤怒、悲伤、爱等情感
- **长期记忆**: 鹦鹉能记住多年未见的人和地方
- **镜像自我认知**: 部分鹦鹉能通过镜像测试（识别镜中的自己），但研究结果不一

### 5.4 情感与社交 / Emotions & Social Bonds

- **悲伤与哀悼**: 鹦鹉会因伴侣或主人的去世而表现出悲伤行为（食欲下降、精神萎靡、鸣叫减少）
- **嫉妒**: 鹦鹉可能对新宠物或新家庭成员表现出嫉妒行为
- **依恋**: 鹦鹉可与主人建立深厚的依恋关系，分离时可能产生焦虑
- **玩耍**: 鹦鹉有玩耍行为，包括倒挂、翻滚、与玩具互动
- **幽默感**: 有观察表明鹦鹉会"恶作剧"（如故意掉落物品看主人捡起）

### 5.5 历史知识 / Historical Knowledge

**古代历史 / Ancient History**:
- 鹦鹉作为宠物饲养的历史至少有 2500 年
- 古埃及和古印度文献中已有鹦鹉的记载
- 古希腊和古罗马贵族以饲养鹦鹉为身份象征
- 亚历山大大帝从印度带回鹦鹉到欧洲

**中世纪与近代 / Medieval & Modern History**:
- 中世纪欧洲，鹦鹉是皇室和贵族的珍贵宠物
- 大航海时代，欧洲探险家从世界各地带回鹦鹉品种
- 19 世纪，鹦鹉贸易繁荣，虎皮鹦鹉被引入欧洲
- 20 世纪，鹦鹉成为全球最受欢迎的宠物鸟之一

**文化象征 / Cultural Symbolism**:
- 在印度教中，鹦鹉是爱神卡玛（Kama）的坐骑
- 在中国文化中，鹦鹉象征忠诚和婚姻幸福（成对出现）
- 在西方文化中，鹦鹉与海盗形象关联（源于小说《金银岛》）
- 鹦鹉在许多原住民文化中具有神圣地位

**著名鹦鹉 / Famous Parrots**:
- **Alex**: 非洲灰鹦鹉，认知研究的明星
- **Snowball**: 葵花凤头鹦鹉，因跳舞视频走红
- **N'kisi**: 非洲灰鹦鹉，据称掌握 950 个词汇，曾展示"通灵"能力（有争议）
- **Puck**: 虎皮鹦鹉，被吉尼斯纪录认定为词汇量最多的鸟（1723 个词）
- **Winston**: 英国首相丘吉尔的非洲灰鹦鹉（据传至今仍活着，已 100 多岁）

### 5.6 趣味事实 / Fun Facts

- 鹦鹉是唯一能用脚"拿"食物送到嘴里的鸟类（大多数鸟类用喙直接啄食）
- 金刚鹦鹉的喙部力量可达 200–300 psi，能咬碎椰子
- 鸮鹦鹉（Kakapo）是世界上唯一不会飞的鹦鹉，也是现存最重的鹦鹉（可达 4 kg）
- 鹦鹉的寿命与体型相关，但小型鹦鹉（如虎皮）的代谢率远高于大型鹦鹉
- 鹦鹉的羽毛颜色由特殊的鹦鹉色素（psittacofulvins）产生，这是鹦鹉独有的色素
- 鹦鹉在野外每天飞行数公里觅食，家养鹦鹉需要足够的飞行或运动空间
- 鹦鹉的智商相当于 3–6 岁儿童，情感复杂度类似
- 鹦鹉能识别镜子中的自己（部分物种）
- 鹦鹉的"笑声"是学习行为，会模仿主人的笑声
- 鹦鹉在夜间可能因噩梦而惊叫（"夜惊"现象）

## 6. 保护现状与法规 / Conservation & Regulations

### 6.1 保护现状 / Conservation Status

- 约 1/3 的鹦鹉物种面临灭绝威胁（IUCN 数据）
- 主要威胁：栖息地丧失、非法宠物贸易、入侵物种、气候变化
- 已灭绝物种：卡罗莱纳长尾鹦鹉（*Conuropsis carolinensis*，1918 年灭绝）、古巴红鹦鹉（*Ara tricolor*，1864 年灭绝）等
- 极危物种：蓝紫金刚鹦鹉（野外数量极少）、鸮鹦鹉（约 200 只）、斯皮克斯金刚鹦鹉（*Cyanopsitta spixii*，野外可能已灭绝）

### 6.2 CITES 附录 / CITES Appendices

《濒危野生动植物种国际贸易公约》（CITES）对鹦鹉贸易有严格管制：

- **附录 I / Appendix I**: 禁止商业贸易的极危物种，包括大多数凤头鹦鹉、部分金刚鹦鹉、非洲灰鹦鹉等
- **附录 II / Appendix II**: 需许可证才能贸易的物种，包括虎皮鹦鹉、玄凤鹦鹉、环颈鹦鹉等常见宠物品种
- **附录 III / Appendix III**: 特定国家管制的物种

### 6.3 购买注意事项 / Purchase Guidelines

**合法购买 / Legal Purchase**:
- 确认鹦鹉来源合法，有 CITES 许可证（如适用）
- 选择信誉良好的繁殖者或救助站
- 避免购买野外捕获的个体（wild-caught）
- 要求提供健康证明和血统记录

**领养替代购买 / Adoption**:
- 许多鹦鹉因主人无法继续饲养而被送交救助站
- 领养是更人道的选择
- 救助站通常会对鹦鹉进行健康检查和行为评估

**品种选择建议 / Species Selection Advice**:
- 初次养鸟者：虎皮鹦鹉、玄凤鹦鹉、牡丹鹦鹉
- 中级养鸟者：环颈鹦鹉、派翁尼斯鹦鹉、吸蜜鹦鹉
- 高级养鸟者：非洲灰鹦鹉、亚马逊鹦鹉、凤头鹦鹉、金刚鹦鹉
- 考虑因素：时间投入、空间、噪音容忍度、预算、长期承诺（寿命）

## 7. 各品种环境需求与监测阈值 / Species Environmental Requirements

> 本章为本系统的核心查询依据：用户在「宠物档案」中录入品种后，系统/智能体可据此推荐该品种的适宜温度、湿度与粉尘浓度区间，并据此设置告警阈值。无论用户饲养何种鹦鹉，本章均应能命中对应数据。
> This chapter is the core reference for the system: after a user selects a species in the pet profile, the system can recommend the appropriate temperature, humidity and dust ranges and configure alarm thresholds accordingly.

### 7.0 通用原则 / General Principles

不同鹦鹉品种原产地气候差异巨大，因此对温度、湿度、粉尘的耐受度各不相同。设置环境参数时应遵循以下通用原则：

1. **温度 / Temperature**：以原产地年均气温为基准，热带低地品种怕冷（低于 18℃ 易应激），高原/温带品种相对耐寒。任何品种都应避免单日温差超过 8℃。
2. **湿度 / Humidity**：折衷鹦鹉等雨林品种需要高湿度（60–80%），干燥会致羽毛脆裂、皮肤脱屑；而虎皮、牡丹等干旱原产物种对低湿度耐受，反而怕持续潮湿（易生曲霉菌）。
3. **粉尘/羽粉 / Dust & Feather Dander**：凤头鹦鹉和灰鹦鹉产生大量羽粉（粉绒羽），自身敏感且污染环境，需高频通风与空气净化。羽粉长期偏高可诱发主人与鹦鹉的呼吸道疾病。
4. **粉尘浓度参考（与本系统 `system_setting` 对齐）**：
   - `< 35 μg/m³`：优 / 正常
   - `35–80 μg/m³`：良 / 中风险（提示通风）
   - `> 80 μg/m³`：差 / 高风险（需立即处置）
5. **光照 / Lighting**：所有品种每日需 10–12 小时全光谱光照（含 UVB）以合成维生素 D3；夜间需 12–14 小时安静黑暗，否则易因激素紊乱出现啄羽和过度尖叫。

### 7.4 粉尘/羽粉特别说明 / Dust & Feather Dander Notes

- **羽粉产生大户（需高频通风 + 空气净化器，建议每日开排风扇 ≥ 2 次）**：所有凤头鹦鹉（葵花、雨伞、摩鹿加、米切氏、戈芬氏）、非洲灰鹦鹉、玄凤鹦鹉。
- **羽粉极少品种（粉尘告警阈值可适当放宽，但需关注湿便）**：吸蜜鹦鹉、折衷鹦鹉、虎皮、牡丹、太平洋。
- **呼吸道敏感品种（粉尘阈值建议下调至 warning=25、danger=60 μg/m³）**：派翁尼斯、灰鹦鹉、凯克、折衷、所有凤头鹦鹉。
- **系统建议**：在「系统设置 → 阈值配置」中，若当前登录宠物为上述敏感品种，应提示用户调低粉尘告警阈值。

### 7.5 温湿度异常的常见后果 / Consequences of Abnormal Temp & Humidity

| 异常 / Abnormality | 品种易感性 / Susceptibility | 后果 / Consequence |
|---|---|---|
| 温度 < 15℃（持续）| 小型鹦鹉、吸蜜、折衷 | 消化停滞、免疫力下降、感冒 |
| 温度 > 32℃（持续）| 大型鹦鹉、灰鹦鹉 | 中暑、张口呼吸、热应激 |
| 单日温差 > 8℃ | 所有品种，尤其虎皮、玄凤 | 应激、打喷嚏、潜伏病爆发 |
| 湿度 < 30%（持续）| 折衷、吸蜜、凯克 | 羽毛脆裂、皮肤脱屑、瘙痒啄羽 |
| 湿度 > 80%（持续）| 虎皮、牡丹、派翁尼斯 | 曲霉菌滋生、呼吸道真菌感染 |
| 粉尘 > 80 μg/m³（持续）| 灰鹦鹉、凤头、玄凤 | 呼吸道炎症、羽粉过敏、主人过敏 |
