## 1. 必须立即就医的危险信号

出现以下任一情况，**不要犹豫，立即联系异宠医院送医**：

- **持续张口呼吸**：尾巴随呼吸上下明显摆动，呼吸有杂音或哨音
- **严重出血**：压迫 5 分钟仍不止血，或喙部、爪部大量出血
- **抽搐、翻倒、无法站立**：可能中毒、脑部疾病或严重代谢紊乱
- **撞击后昏迷或失去平衡**：可能脑震荡或颈椎损伤
- **误食有毒物后**：牛油果、巧克力、洋葱、重金属、有毒植物等
- **腹部明显肿胀、蹲伏喘息**：可能蛋难产或内脏疾病
- **呕吐或反复反流**
- **粪便异常持续**：血便、纯白色、黑色柏油便、持续水便
- **完全拒食超过 12 小时**（小型鹦鹉）或 24 小时（中大型）
- **特氟龙气体暴露后呼吸异常**：哪怕几分钟也需紧急就医

> 鹦鹉代谢快，小型鹦鹉（虎皮、牡丹）拒食 24 小时即可因低血糖危及生命，不可拖延。

## 2. 家庭临时处置总原则

1. **保温**：生病或受伤的鹦鹉体温调节能力下降，需保温至 28–30℃（用保温灯，留出冷区）。
2. **隔离**：移至安静、暗光的小笼（医院笼），减少应激。
3. **补水**：提供电解质水（药店口服补液盐按说明稀释），防止脱水。
4. **不强行喂食**：虚弱鹦鹉强行喂食可能呛入气管，仅提供易消化软食。
5. **观察记录**：用手机拍照/录像记录症状、粪便、呼吸状态，供兽医诊断。
6. **联系兽医**：处置同时联系异宠医院，告知症状并预约。

### 2.1 切忌事项

- ❌ 不要用冰水或热水直接接触鹦鹉
- ❌ 不要自行喂食人用药物（对乙酰氨基酚、布洛芬等对鹦鹉有毒）
- ❌ 不要把虚弱鹦鹉放在强光或嘈杂环境
- ❌ 不要抓握过紧（鹦鹉胸腔脆弱，窒息风险）
- ❌ 不要剪开放性伤口上的血羽

## 3. 呼吸急促与张口呼吸

### 3.1 判断

健康鹦鹉呼吸应平稳无声。张口呼吸、尾巴随呼吸上下摆动（尾摆）、呼吸有哨音或湿啰音，均为呼吸窘迫信号。

### 3.2 常见原因

- 空气污染（粉尘、烟雾、特氟龙气体、香薰）
- 呼吸道感染（细菌、真菌如曲霉菌、衣原体鹦鹉热）
- 气囊螨（常见于虎皮、玄凤）
- 中暑
- 心脏疾病

### 3.3 家庭处置

1. **立即转移**到空气清新、温度适宜（25℃左右）的房间。
2. **加湿**：用喷雾瓶向空气中喷水（避免直喷鹦鹉），湿度维持 50–60%，有助于缓解呼吸道干燥。
3. **保温**至 28–30℃。
4. **隔离静养**，避免打扰。
5. **记录呼吸频率与表现**（录像）。
6. **立即联系兽医**：呼吸问题恶化快，不可在家观察过夜。

### 3.4 特氟龙气体暴露

若近期使用过不粘锅、烤箱自清洁、电饭煲排气等，怀疑特氟龙气体暴露：

1. **立即开窗通风 + 转移鹦鹉到其他房间**。
2. **即使鹦鹉看起来正常也需就医**：肺损伤可能在数小时后恶化。
3. 详见 [告警应急处理.md](告警应急处理.md) 第 8 节。

## 4. 中暑

### 4.1 判断

环境温度 >32℃，鹦鹉出现：

- 张口呼吸、翅膀张开
- 精神萎靡、步态不稳
- 严重时抽搐、昏迷

### 4.2 家庭处置

1. **立即转移**到阴凉通风处（25℃左右）。
2. **喷雾降温**：用喷雾瓶向鹦鹉羽毛喷**微温水**（30℃左右，**不可用冰水**，会导致血管收缩休克）。
3. **脚部降温**：用湿毛巾包裹脚趾辅助散热。
4. **提供凉水**（非冰水）饮用。
5. **轻柔扇风**辅助降温。
6. **立即送医**：中暑可致多器官损伤，即使缓解也需检查。

### 4.3 预防

- 夏季笼舍避开阳光直射，温度控制在 28℃以下。
- 提供洗澡水。
- 主人不在家时开启空调（避免直吹）。
- 系统温度告警阈值建议设在 30℃。

## 5. 失温

### 5.1 判断

环境温度过低，鹦鹉出现：

- 炸毛、缩成一团
- 颤抖、步态僵硬
- 精神萎靡、反应迟钝
- 严重时昏迷

### 5.2 家庭处置

1. **缓慢加温**：用保温灯或加热垫，**缓慢**升至 28–30℃。**切忌快速高温加热**（可致休克）。
2. **隔离静养**。
3. **提供温热电解质水**（30℃左右）。
4. **观察恢复**：鹦鹉恢复活动后再放回原笼。
5. 失温常伴随疾病，恢复后建议体检。

### 5.3 预防

- 冬季笼舍保温 18℃以上。
- 避免冷风直吹与温差骤变。
- 夜间遮盖笼布。

## 6. 出血

### 6.1 血羽出血（最常见）

鹦鹉换羽期，新生长的羽毛（血羽）含血管，折断会出血。

1. **拔除断羽**：用止血钳或镊子夹住羽毛根部（贴近皮肤），沿生长方向**果断拔出**。
2. **按压止血**：拔出后用纱布按压 3–5 分钟。
3. **止血粉**：可用止血粉（如 Kwik Stop）辅助。
4. 拔除困难或出血不止需就医。

### 6.2 喙部/爪部出血

1. 干净纱布按压止血 5 分钟。
2. 止血粉辅助。
3. 喙部出血量大或持续不止需就医（可能伤及深层组织）。

### 6.3 大量出血

- 腹部、胸部大量出血：**用纱布压迫 + 立即送医**，途中持续按压。
- 鹦鹉失血耐受差，少量失血即可能危及生命。

## 7. 中毒

### 7.1 常见中毒源

| 中毒源 | 症状 | 处置 |
|---|---|---|
| 牛油果 / Persin | 呼吸困难、心力衰竭、猝死 | 立即送医，无特效解毒 |
| 巧克力 / 可可碱 | 心律不齐、抽搐、呕吐 | 立即送医 |
| 洋葱、大蒜 | 溶血性贫血、虚弱、尿血 | 立即送医 |
| 重金属（铅、锌）| 神经症状、呕吐、绿色便 | 立即送医，可能需螯合治疗 |
| 特氟龙气体 | 急性呼吸窘迫、肺水肿 | 立即通风 + 送医 |
| 有毒植物（杜鹃、常春藤等）| 呕吐、腹泻、神经症状 | 送医，携带植物样本 |
| 酒精 | 中枢抑制、昏迷 | 立即送医 |
| 杀虫剂、农药 | 神经症状、流涎 | 立即送医，携带毒物包装 |

### 7.2 通用处置

1. **立即移除毒源**：移走有毒食物/物品，通风。
2. **不要催吐**：鹦鹉催吐风险大，可能误吸。
3. **记录摄入物与时间**：拍照包装、估计摄入量。
4. **立即送医**：联系异宠医院，告知毒物类型。
5. **保留毒物样本**：连同包装带去兽医处。

## 8. 抽搐与癫痫

### 8.1 判断

鹦鹉倒地、翅膀扑打、腿部划动、头部后仰、意识丧失。

### 8.2 常见原因

- 中毒（重金属、巧克力、杀虫剂）
- 严重低血糖（多见小型鹦鹉拒食后）
- 脑部疾病、感染
- 头部外伤
- 严重缺钙（多见于灰鹦鹉）

### 8.3 家庭处置

1. **立即隔离**到铺有软垫的小笼，移开硬物与栖木，避免撞伤。
2. **暗光静养**：用布遮盖笼子，减少刺激。
3. **不要强行抱握**或向口中灌水（呛吸风险）。
4. **记录抽搐表现与持续时间**（录像）。
5. **抽搐停止后**：保温 28–30℃，提供葡萄糖水（小型鹦鹉低血糖可用 5% 葡萄糖水滴喙缘）。
6. **立即送医**。

## 9. 撞伤与骨折

### 9.1 撞伤（飞行撞玻璃/墙）

1. **隔离静养**：放入小笼，降低栖木，避免再次飞行。
2. **观察意识与平衡**：若昏迷、歪头、无法站立，可能脑震荡，立即送医。
3. **保温**。
4. **观察 24 小时**：即使表面正常也需观察，内伤可能延迟出现。

### 9.2 骨折（腿/翅）

1. **限制活动**：小笼静养，移除栖木，垫软垫。
2. **不要自行固定**：不当固定会加重损伤。
3. **保温**。
4. **立即送医**：骨折需专业处理，拖延会导致无法复位。

## 10. 蛋难产

### 10.1 判断

雌性鹦鹉（尤其玄凤、虎皮、牡丹）出现：

- 腹部肿胀、蹲伏、用力
- 精神差、食欲下降
- 排便困难、粪便细小
- 呼吸急促

### 10.2 家庭处置

1. **保温加湿**：28–30℃，湿度 60%，温暖湿润有助于产蛋。
2. **提供钙质**：电解质水中加钙（液体钙按鸟体重）。
3. **隔离静养**，避免打扰。
4. **不要挤压腹部**。
5. **数小时未产出立即送医**：蛋难产可致命，需兽医助产或手术。

### 10.3 预防

- 提供充足钙质（墨鱼骨、矿物质块）。
- 控制产蛋频率（减少筑巢刺激）。
- 雌性易患，建议定期体检。

## 11. 拒食与虚弱

### 11.1 判断

鹦鹉完全不吃食物超过 12 小时（小型）或 24 小时（中大型），精神萎靡、炸毛、体重下降。

### 11.2 家庭处置

1. **保温** 28–30℃。
2. **提供易消化软食**：温热小米粥、电解质水、5% 葡萄糖水。
3. **必要时灌食**：用细管轻柔灌入流食（无经验者不建议，呛吸风险）。
4. **记录体重**：用克秤称重，持续下降需紧急就医。
5. **立即送医**：拒食是严重疾病信号，需排查原因。

## 12. 夜惊

### 12.1 判断

夜间鹦鹉突然惊叫、扑翅、撞笼，常因噩梦或外界声响。

### 12.2 家庭处置

1. **不要开强光**：开小夜灯，轻声安抚。
2. **检查有无受伤**：夜惊撞笼可致撞伤或血羽折断。
3. **恢复后让其继续休息**。

### 12.3 预防

- 夜间遮盖笼布（留通风口）。
- 笼舍远离噪音源（电视、马路）。
- 留小夜灯（避免完全黑暗）。
- 系统蜂鸣器夜间应关闭自动联动，避免惊吓。

## 13. 家庭急救箱

建议常备以下物品：

| 类别 | 物品 | 用途 |
|---|---|---|
| 止血 | 止血粉（Kwik Stop）、纱布、棉签 | 出血止血 |
| 保温 | 保温灯、加热垫、温度计 | 失温、生病保温 |
| 补液 | 口服补液盐、5% 葡萄糖、电解质水 | 脱水、低血糖 |
| 消毒 | 碘伏、生理盐水 | 伤口消毒 |
| 工具 | 止血钳、镊子、鸟用指甲剪 | 拔血羽、修剪 |
| 隔离 | 备用小笼（医院笼）、软垫 | 隔离静养 |
| 记录 | 手机、克秤 | 拍照记录、称重 |
| 联系 | 异宠医院电话、24h 急诊电话 | 送医 |

## 14. 送医前准备

1. **联系医院**：提前电话告知症状，确认接收。
2. **准备运输笼**：小笼 + 软垫 + 保温（冬季途中保温）。
3. **携带资料**：
   - 鹦鹉档案（品种、年龄、体重、病史，可从系统导出）
   - 症状照片/录像
   - 粪便样本（用保鲜袋装新鲜粪便）
   - 怀疑中毒时携带毒物包装
4. **保温运输**：途中维持 25–28℃。
5. **避免应激**：运输笼遮布，减少视觉刺激。

## 15. 品种特殊注意事项

| 品种 | 特殊急救注意 |
|---|---|
| 非洲灰鹦鹉 | 易低钙抽搐，急救时注意补钙；易啄羽自残 |
| 虎皮、牡丹 | 体型小、代谢快，拒食 24 小时即危险，需紧急补葡萄糖 |
| 玄凤 | 雌性易蛋难产；易患气囊螨（呼吸杂音）|
| 吸蜜鹦鹉 | 湿便正常，勿误判；饮食特殊，急救灌食需用花蜜配方 |
| 折衷鹦鹉 | 高湿度需求，急救环境需 60%+ 湿度 |
| 凤头鹦鹉 | 应激反应强烈，处置需格外安静；羽毛粉尘大 |
| 金刚鹦鹉 | 喙力强，处置需戴手套防咬伤 |

> 各品种环境敏感度详见 [parrot-knowledge-base.md](parrot-knowledge-base.md) 第 7 章。

## 16. 急救流程速查

```
发现异常
  │
  ├─ 是否符合第1节危险信号？ ──是──> 立即送医（途中按对应章节处置）
  │
  └─ 否 ──> 第2节总原则处置（保温+隔离+补水+观察+记录）
              │
              ├─ 30 分钟内改善 ──> 继续观察，预约常规体检
              │
              └─ 无改善或恶化 ──> 送医
```
