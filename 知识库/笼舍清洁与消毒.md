# 笼舍清洁与消毒 SOP

> 笼舍清洁是鹦鹉健康的基础。粪便、食物残渣、羽粉堆积是细菌、真菌（尤其曲霉菌）滋生的温床，也是系统粉尘告警的主要来源之一。本 SOP 规范每日、每周、每月清洁任务与消毒剂选择，确保笼舍卫生安全。

## 1. 清洁原则

1. **从上到下**：先清洁高处栖木、玩具，再清笼底，避免二次污染。
2. **干湿分离**：先清扫固体废物（粪便、羽毛、食物残渣），再湿擦消毒。
3. **彻底冲洗**：消毒剂残留对鹦鹉有毒，消毒后必须用清水彻底冲洗并晾干。
4. **鹦鹉隔离**：深度清洁与消毒时，将鹦鹉转移到备用笼，避免吸入清洁剂挥发物。
5. **通风**：清洁与消毒全程开窗通风，避免鹦鹉吸入挥发性气体。
6. **专物专用**：鹦鹉清洁工具（刷子、抹布、喷雾瓶）与家用分开，避免交叉污染。

## 2. 清洁频率总览

| 任务 | 频率 | 耗时 |
|---|---|---|
| 更换饮水、清理食盆 | 每日 1–2 次 | 5 分钟 |
| 清理笼底托盘与垫料 | 每日 1 次 | 5 分钟 |
| 擦拭栖木与食盆 | 每日 1 次 | 5 分钟 |
| 局部擦拭粪便 | 每日按需 | 2 分钟 |
| 清洗玩具、秋千 | 每周 1 次 | 15 分钟 |
| 消毒笼体与配件 | 每周 1 次 | 30 分钟 |
| 更换/清洗垫料 | 每周 1 次 | 10 分钟 |
| 深度清洁（拆卸全笼）| 每月 1 次 | 1–2 小时 |
| 更换栖木（磨损）| 每季度 / 按需 | 15 分钟 |
| 检查笼具锈蚀 | 每季度 | 10 分钟 |

> 羽粉多品种（灰鹦鹉、凤头、玄凤）建议频率提高一档。

## 3. 每日清洁

### 3.1 任务清单

- [ ] 倒掉旧水，用刷子清洗水盆，换新鲜饮水。
- [ ] 清理食盆，丢弃剩食（尤其易腐蔬果），换新粮。
- [ ] 抽出笼底托盘，刮除粪便与食物残渣，更换垫料（报纸/厨房纸巾/专用垫料）。
- [ ] 检查栖木，若有点状粪便及时用湿布擦拭。
- [ ] 观察粪便形态（颜色、成形度），异常记录。
- [ ] 检查鹦鹉羽毛、精神、行为状态。

### 3.2 注意事项

- 蔬果剩余超过 2 小时（夏季 1 小时）应丢弃，防止变质。
- 水盆若有食物掉落或粪便污染，立即更换。
- 使用厨房纸巾作为垫料便于每日更换与观察粪便（报纸油墨有争议，建议优先纸巾）。

## 4. 每周清洁

### 4.1 任务清单

- [ ] 将鹦鹉转移到备用笼。
- [ ] 拆卸玩具、秋千、梯子、食盆、水盆。
- [ ] 用温肥皂水刷洗玩具与配件，清水冲洗，晾干。
- [ ] 用宠物专用消毒剂或白醋水擦拭笼体（栅栏、托盘、底盘）。
- [ ] 清水彻底擦拭，去除消毒剂残留。
- [ ] 晾干或用干净布擦干。
- [ ] 更换或清洗垫料（若用布垫料）。
- [ ] 检查栖木磨损，必要时更换。
- [ ] 清洁笼周围地面与墙面（羽粉会扩散）。

### 4.2 玩具清洁

- 啃咬玩具（木枝、皮革、纸板）：耗材类，磨损严重直接更换，不宜水洗（易发霉）。
- 塑料/金属玩具：温肥皂水刷洗，清水冲净，晾干。
- 绳子玩具：若沾粪便难以清洁，建议更换（细菌藏匿）。
- 觅食玩具：拆卸清洗每个部件，确保无食物残留。

## 5. 每月深度清洁

### 5.1 任务清单

- [ ] 将鹦鹉转移到备用笼，移到其他房间（避免吸入挥发物）。
- [ ] 完全拆卸笼具（栖木、玩具、食盆、托盘、底盘）。
- [ ] 用吸尘器（带 HEPA）吸除笼体与周围羽粉。
- [ ] 用温肥皂水全面刷洗笼体，重点角落与接缝。
- [ ] 用宠物专用消毒剂浸泡配件 10–15 分钟。
- [ ] 清水彻底冲洗所有部件（关键步骤）。
- [ ] 阳光下晾晒消毒（紫外线杀菌），或用干净布擦干。
- [ ] 检查笼具锈蚀、涂层剥落（铅锌中毒风险），必要时更换。
- [ ] 检查并更换磨损栖木。
- [ ] 重新组装，放回鹦鹉前确保无清洁剂气味。

### 5.2 笼周围环境

- 清洁笼子所在墙面、地面、附近家具（羽粉会附着）。
- 清洗窗帘、笼布。
- 清洁或更换空气净化器滤网（每 3–6 个月）。

## 6. 消毒剂选择

### 6.1 安全消毒剂 ✅

| 消毒剂 | 用法 | 适用场景 |
|---|---|---|
| 白醋 + 水（1:1）| 擦拭后清水冲 | 日常温和消毒、除味 |
| 宠物专用消毒剂（如 F10SC）| 按说明稀释，擦拭/浸泡 | 全面消毒，对鸟安全 |
| 温肥皂水 | 刷洗 | 日常去污 |
| 阳光紫外线 | 晾晒 2–4 小时 | 配件天然杀菌 |
| 沸水 | 浸泡耐热配件 | 物理消毒 |

> F10SC 是兽医推荐的鸟类安全消毒剂，对细菌、病毒、真菌均有效，稀释后对鹦鹉低毒，但仍需冲洗。

### 6.2 禁用消毒剂 ❌

| 消毒剂 | 风险 |
|---|---|
| 含氯漂白剂（高浓度）| 刺激呼吸道，残留有毒；若必须用需极稀 + 彻底冲洗 + 充分通风 |
| 苯酚类（如滴露、来苏儿）| 对猫狗鸟均剧毒，**绝对禁用** |
| 氨水 | 刺激呼吸道，有毒 |
| 酒精（大面积）| 挥发刺激，易燃 |
| 强酸/强碱清洁剂 | 腐蚀、有毒 |
| 含香精的清洁剂 | 香精对鹦鹉呼吸道刺激 |

### 6.3 使用原则

- 优先物理清洁（刷洗 + 热水 + 阳光），化学消毒为辅。
- 任何消毒剂使用后必须用清水彻底冲洗 2–3 遍。
- 消毒全程鹦鹉不在场，通风至无气味再放回。
- 不同消毒剂不可混合（如漂白剂 + 酒精产生剧毒气体）。

## 7. 消毒操作步骤

### 7.1 准备

1. 转移鹦鹉到备用笼，移到其他房间。
2. 开窗通风。
3. 准备：温肥皂水、消毒剂、清水、刷子、抹布、手套、口罩（保护自己免受羽粉与消毒剂刺激）。

### 7.2 清洁

1. 干扫：用吸尘器或刷子清除固体废物与羽粉。
2. 湿洗：温肥皂水刷洗笼体与配件，去除有机物（消毒剂在有机物存在时效力下降）。
3. 清水冲洗。

### 7.3 消毒

1. 按说明稀释消毒剂（如 F10SC 1:250）。
2. 喷洒或浸泡笼体与配件，作用 10–15 分钟。
3. 角落、接缝、栖木底端重点处理。

### 7.4 冲洗与干燥

1. 清水彻底冲洗 2–3 遍，确保无残留。
2. 阳光下晾晒或用干净布擦干。
3. **确认无气味**后放回鹦鹉。

## 9. 安全注意事项

1. **鹦鹉隔离**：消毒全程鹦鹉不在场，避免吸入挥发物。
2. **彻底冲洗**：消毒剂残留是中毒常见原因，冲洗不可省略。
3. **通风**：清洁后通风 ≥30 分钟再放回鹦鹉。
4. **个人防护**：操作者戴手套、口罩（羽粉与消毒剂均刺激呼吸道）。
5. **笼具检查**：发现锈蚀、涂层剥落立即更换（铅锌中毒）。
6. **食物防污染**：清洁剂远离食物与食盆。
7. **备用笼**：始终保持一个备用小笼，用于清洁时安置鹦鹉。
8. **记录**：在病历或日志中记录深度清洁日期，便于追溯。

## 10. 与系统的衔接

### 10.1 粉尘告警与清洁

- 笼底粪便与羽粉堆积是粉尘告警的常见来源之一。
- 收到 `smoke_high`（粉尘）告警时，排查清单应包含「笼底是否需清洁」。
- 清洁笼底后，粉尘浓度通常显著下降，可作为处置验证。

### 10.2 清洁记录归档

- 建议在「记账本」或「病历」中记录深度清洁日期（可作为养护日志）。
- 后续可在 `pet_media_record` 中保存笼舍清洁前后对比照片。

### 10.3 湿度与清洁

- 湿度告警（`humidity_high`）时，排查笼底是否积水或垫料潮湿。
- 潮湿垫料是曲霉菌温床，需立即更换。

### 10.4 联动设备与清洁

- 排风扇在清洁后可手动开启，加速干燥与通风。
- 空气净化器在清洁后开启，过滤扬起的羽粉。
