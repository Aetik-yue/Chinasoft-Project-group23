# 急救箱与常见意外处理

意外发生时，正确的前期处理能为就医争取时间。本篇讲怎么准备一个鹦鹉急救箱，以及几种常见意外的处理原则。

## 急救箱里放什么

- **止血粉**：处理出血（剪指甲、外伤）。
- **消毒液**：F10 或稀释碘伏（不是酒精）。
- **纱布、棉签**：清洁和包扎。
- **小毛巾**：保定（包裹）用。
- **加热垫/暖宝宝**：失温时保暖（隔布使用）。
- **外出笼**：紧急转移用。
- **电解质粉**：腹泻、中暑后补水。
- **针筒（无针头）**：喂水喂药用。
- **止血钳、小剪刀**：处理缠绕物。
- 异宠医院联系方式、24 小时急诊电话。

## 出血处理

1. 找到出血点，判断是甲出血、羽出血还是外伤。
2. 用纱布按压出血处 2-3 分钟。
3. 按压不止就撒止血粉再按。
4. 羽出血（剪到血羽）要拔掉整根血羽才能止住，拔时用止血钳夹根部顺向拔出。
5. 出血量大或止不住，立刻就医。

## 撞伤 / 摔伤

- 轻轻移到安静、温暖的笼子或外出笼。
- 不要强行检查，避免二次伤害。
- 观察是否能站立、是否有歪头、翅膀下垂。
- 即使表面没事也建议就医，内伤早期看不出来。
- 24 小时内限制活动，避免剧烈飞行。

## 缠绕窒息

- 脚或脖子被玩具绳缠住，先用剪刀小心剪断绳子。
- 不要硬扯，会勒更紧。
- 解开后检查有无勒痕、淤血。
- 严重勒伤就医检查血液循环。

## 中毒

参考《鹦鹉常见有毒食物清单》：

1. 立刻移走毒物，记录吃了什么、多少、何时。
2. 不要自行催吐。
3. 带上残留食物/包装，立刻就医。
4. 途中保持温暖、安静。

## 中暑

参考《夏季防暑与冬季保暖》：

1. 移到阴凉通风处。
2. 喷雾轻喷脚掌和翅膀内侧。
3. 不要冰水浇身。
4. 恢复后就医检查。

## 失温

- 鹦鹉炸毛、发抖、精神萎靡可能是失温。
- 移到温暖房间（25-28℃）。
- 用毛巾包裹或加热垫隔布加温。
- 不要用吹风机高温直吹。
- 体温回升后仍建议就医，排查病因。

## 油污染羽毛

- 鹦鹉沾到食用油/机油会影响保温和梳理。
- 不要用溶剂清洗。
- 用玉米淀粉或爽身粉吸附油脂，反复轻搓。
- 严重油污必须由兽医处理，自己清洗可能伤皮肤。

## 就医前的准备

- 用外出笼带，里面垫毛巾。
- 带上平时的粪便样本（如拉稀）。
- 记录症状出现时间、变化。
- 带上正在用的药物/营养品清单。
- 提前打电话确认医院营业。

## 平时预防

- 定期安全检查笼子和玩具。
- 关好门窗，避免飞撞。
- 厨房烹饪时移开鹦鹉。
- 收好小零件、绳子、有毒植物。

> 小贴士：把急救箱和急救电话贴在一起，全家都知道在哪；每半年检查一次药品保质期。
