# 夏季防暑与冬季保暖

鹦鹉对温度变化很敏感。相比绝对的温度高低，**温差过大**和**直吹风**更容易让它们生病。大多数小型鹦鹉（如虎皮、玄凤、小太阳）最适宜的温度是 20-26℃。本篇按季节讲怎么管理环境。

## 适宜环境

- 温度：20-26℃
- 湿度：40-60%
- 一天内温差尽量不超过 8℃

同时要保证空气流通，但不能让风直接吹到笼子。建议在笼子附近放一个温湿度计，比凭感觉判断可靠。

## 夏季防暑

- 避免阳光直射笼子，可用遮光帘或移到阴凉通风处。
- 提供浅水盆让它自己洗澡降温，水深不超过脚踝。
- 可在笼旁放一盆冰水辅助降温（不要直接对着鹦鹉吹冰）。
- **不要**对着空调或风扇直吹。
- 夏天食物容易变质，食盒里的湿料（水果、蔬菜）2 小时内吃不完要倒掉。

### 中暑征兆

如果发现鹦鹉张嘴呼吸、翅膀张开、精神萎靡，可能是中暑：

1. 立刻转移到阴凉通风处。
2. 用喷雾轻喷脚掌和翅膀内侧降温。
3. 不要直接用冰水浇身，温差过大会休克。
4. 精神恢复后仍建议就医检查。

## 冬季保暖

- 保温灯或加热垫应固定在笼子**外侧**，防止烫伤。
- 夜间可用厚笼衣盖住，但要留一点通风口。
- **不要**把笼子直接放在暖气出风口，会过于干燥。
- **不要**用电热毯裹住笼子，容易过热或缺氧。
- 保证室内湿度，必要时用加湿器（不要对着笼子直喷）。

### 保暖设备选择

| 设备 | 适用 | 注意 |
|---|---|---|
| 陶瓷保温灯 | 夜间恒温 | 必须配灯罩，防烫伤 |
| 加热垫 | 底部局部加温 | 固定在笼外，留冷区 |
| 笼衣 | 夜间保温 | 留通风口 |
| 室内空调 | 整体控温 | 避免直吹 |

## 换季注意

春秋换季时气温波动大，是感冒高发期：

- 这段时间要更频繁地观察精神状态和排泄。
- 必要时减少开窗时间，避免穿堂风。
- 换羽期体能消耗大，适当补充营养。
- 注意早晚温差，晚上提前盖笼衣。

## 梅雨季防潮

- 湿度长期高于 70% 易滋生霉菌和呼吸道问题。
- 使用除湿机或空调除湿模式。
- 笼底垫纸勤换，防止发霉。
- 食物密封保存，避免受潮变质。

> 小贴士：买一个带记录功能的温湿度计，能回看一天的最高最低值，更容易发现隐患。
