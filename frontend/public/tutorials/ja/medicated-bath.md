# 薬浴前後の保温ポイント

薬浴は、ハジラミ、ふけ、真菌の問題などの治療に用いられます。しかし、インコは体温が高く、羽の保温性も高いため、体がぬれた後に冷えると、風邪や呼吸器疾患を起こしやすくなります。保温は薬浴を成功させる鍵です。ここでは、薬浴前、薬浴中、薬浴後の注意点を順に説明します。

## 薬浴が必要なとき

健康なインコの日常的な水浴びは、清水の霧吹きや浅い水入れで十分です。薬浴を行うのは、次の状況に当てはまり、獣医師から勧められた場合に限ります。

- 毛引きが頻繁で、ふけが目立つ。
- 皮膚や羽に異常な分泌物がある。
- ハジラミやダニなどの外部寄生虫と診断された。
- 獣医師から症状に合った薬浴方法を処方された。

「羽をつややかにする」ためだけに薬浴を繰り返さないでください。使いすぎると、羽が本来持つ天然の油分を損ないます。

## 薬浴前の準備

- 室温を25～28℃に保ち、扇風機やエアコンの風が直接当たらないようにします。
- 約30～35℃のぬるま湯を用意します。手首の内側で温度を確かめ、温かいものの熱くない程度にします。
- 獣医師が指示した薬液濃度を守り、自己判断で濃くしないでください。
- 浅い容器を選び、水位はすねより下にして、水を吸い込む事故を防ぎます。
- ドライヤー、タオル、保温ランプをあらかじめ用意し、薬浴後すぐに対処できるようにします。

## 薬浴中

- インコ自身が水の中に立って水浴びできるようにし、無理に押さえ付けたり、頭を浸したりしないでください。
- 時間は5～10分にとどめます。長すぎると体温が下がります。
- 強く嫌がる、震える、口を開けて呼吸する様子があれば、直ちに中止します。
- 薬液が目や鼻孔に入らないようにします。必要なら、浸す代わりに低い位置から小型の霧吹きで吹きかけます。

## 薬浴後の保温（最も重要な手順）

1. まず柔らかいタオルで表面の水分を吸い取ります。強くこすらないでください。
2. ドライヤーの低温設定で離れた場所から乾かします。同時に手で羽をかき分け、内側まで完全に乾かします。
3. 乾いた後は、暖かく静かな場所で1～2時間休ませます。
4. 薬浴後2時間は、風の当たる場所やエアコンの効いた部屋へ移さないでください。
5. その日は二度目の水浴びをさせないでください。

## 注意が必要な状態

- 薬浴後30分を超えて震えが続く。
- くしゃみや鼻水が出る。
- 元気がなく、食べない。
- 皮膚が赤くなる、または新たなふけが出る。

> ヒント：冬はできる限り薬浴を避けてください。どうしても必要な場合は、一日の中で最も暖かい正午頃を選び、室温を28℃以上に上げて行います。
