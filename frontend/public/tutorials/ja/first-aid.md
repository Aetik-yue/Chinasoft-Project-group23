# 救急箱とよくある事故への対処

事故が起きたとき、適切な初期対応を行えば、受診までの貴重な時間を確保できます。ここでは、インコ用救急箱の準備方法と、よくある事故への基本的な対処を説明します。

## 救急箱に入れる物

- **止血剤**：爪切りや外傷による出血に使用します。
- **消毒液**：F10または希釈したポビドンヨード（アルコールではありません）。
- **ガーゼ、綿棒**：洗浄と包帯処置に使用します。
- **小さなタオル**：保定して包むために使います。
- **ヒーティングパッド・使い捨てカイロ**：低体温時の保温用（布越しに使います）。
- **キャリーケージ**：緊急時の移動用です。
- **電解質パウダー**：下痢や熱中症後の水分補給に使います。
- **シリンジ（針なし）**：水や薬を与えるために使います。
- **止血鉗子、小型はさみ**：絡み付いた物を除去します。
- エキゾチックアニマル病院の連絡先と、24時間対応の救急電話番号。

## 出血への対処

1. 出血部位を見つけ、爪、羽、外傷のどこから出血しているか確認します。
2. ガーゼで出血部位を2～3分間圧迫します。
3. 圧迫しても止まらなければ、止血剤を付けて再び圧迫します。
4. 成長中の血羽を切って出血した場合は、血羽を根元から抜かなければ止血できないことがあります。止血鉗子で根元をつかみ、羽の向きに沿って抜きます。
5. 出血量が多い、または止まらない場合は、直ちに受診してください。

## 衝突・落下によるけが

- 静かで暖かいケージまたはキャリーへ、そっと移します。
- 二次的なけがを防ぐため、無理に全身を調べないでください。
- 立てるか、首が傾いていないか、翼が垂れていないか観察します。
- 外見上は問題がなくても、初期には内傷が分からないため受診をおすすめします。
- 24時間は活動を制限し、激しく飛ばせないでください。

## 絡まりによる窒息

- 足や首におもちゃのひもが絡んだ場合、まずはさみで慎重に切ります。
- 無理に引っ張ると、さらに強く締まるため避けてください。
- 外した後、締め付け跡や内出血がないか確認します。
- 強く締め付けられた場合は受診し、血流を確認してもらいます。

## 中毒

「インコに有毒な代表的食品一覧」を参照してください。

1. 直ちに毒物を取り除き、何を、どれだけ、いつ食べたか記録します。
2. 自分で吐かせないでください。
3. 残った食べ物や包装を持ち、直ちに受診します。
4. 移動中は暖かく静かな状態を保ちます。

## 熱中症

「夏の暑さ対策と冬の保温」を参照してください。

1. 日陰で風通しのよい場所へ移します。
2. 足裏と翼の内側へ霧吹きを軽くかけます。
3. 氷水を全身にかけないでください。
4. 回復した後も受診し、状態を確認してもらいます。

## 低体温

- 羽を逆立てる、震える、元気がない場合は、低体温の可能性があります。
- 暖かい部屋（25～28℃）へ移します。
- タオルで包むか、布越しにヒーティングパッドを当てて温めます。
- ドライヤーの高温風を直接当てないでください。
- 体温が戻っても、原因を調べるため受診をおすすめします。

## 油で羽が汚れた場合

- 食用油や機械油が付くと、保温と羽繕いに支障が出ます。
- 溶剤で洗わないでください。
- コーンスターチまたはベビーパウダーに油を吸着させ、優しく何度かもみ込みます。
- ひどい油汚れは獣医師による処置が必要です。自分で洗うと皮膚を傷めることがあります。

## 受診前の準備

- 底にタオルを敷いたキャリーで連れて行きます。
- 下痢などがある場合は、普段のフンのサンプルを持参します。
- 症状が始まった時刻と変化を記録します。
- 使用中の薬やサプリメントの一覧を持参します。
- 事前に電話し、病院が診療中か確認します。

## 日頃の予防

- ケージとおもちゃを定期的に安全点検します。
- 扉と窓を閉め、衝突や飛び出しを防ぎます。
- キッチンで調理するときはインコを離します。
- 小部品、ひも、有毒植物を片付けます。

> ヒント：救急箱と救急連絡先を同じ場所に置き、家族全員が場所を把握できるようにしてください。半年に一度、薬の使用期限を確認します。
