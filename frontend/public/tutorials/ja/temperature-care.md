# 夏の暑さ対策と冬の保温

インコは温度変化に敏感です。絶対的な温度の高低よりも、**大きな寒暖差**と**風が直接当たること**のほうが病気の原因になりやすい傾向があります。セキセイインコ、オカメインコ、ウロコインコなど、ほとんどの小型インコに適した温度は20～26℃です。ここでは、季節ごとの環境管理を説明します。

## 適した環境

- 温度：20～26℃
- 湿度：40～60％
- 一日の寒暖差はできるだけ8℃以内

空気の流れは確保しながら、ケージへ風が直接当たらないようにします。感覚だけで判断するより、ケージの近くに温湿度計を置くほうが確実です。

## 夏の暑さ対策

- ケージに直射日光が当たらないよう、遮光カーテンを使うか、日陰で風通しのよい場所へ移します。
- 自分で水浴びして体温を下げられるよう、足首より浅い水入れを用意します。
- 補助的な冷却としてケージのそばに氷水を入れた容器を置くこともできますが、冷気を直接インコへ当てないでください。
- エアコンや扇風機の風を**直接当てないでください**。
- 夏は食べ物が傷みやすいため、餌入れの果物や野菜など水分の多い餌は、2時間以内に食べ切らなければ捨てます。

### 熱中症の兆候

口を開けて呼吸する、翼を広げる、元気がないといった状態は、熱中症の可能性があります。

1. 直ちに日陰で風通しのよい場所へ移します。
2. 足裏と翼の内側へ霧吹きを軽くかけ、体温を下げます。
3. 氷水を直接全身へかけないでください。急激な温度差でショックを起こします。
4. 元気が戻った後も、受診して確認してもらうことをおすすめします。

## 冬の保温

- 保温ランプやヒーティングパッドは、やけどを防ぐためケージの**外側**に固定します。
- 夜は厚手のケージカバーを掛けられますが、換気口を少し残します。
- 乾燥しすぎるため、暖房の吹き出し口の前へケージを直接置かないでください。
- 過熱や酸欠を招くため、電気毛布でケージを包まないでください。
- 室内の湿度を保ち、必要に応じて加湿器を使います。ただしケージへ直接噴霧しないでください。

### 保温器具の選び方

| 器具 | 用途 | 注意点 |
|---|---|---|
| セラミックヒーター | 夜間の保温 | やけど防止のカバーが必須 |
| ヒーティングパッド | 底部の部分保温 | ケージ外側に固定し、暖めない区域も残す |
| ケージカバー | 夜間の保温 | 換気口を残す |
| 室内エアコン | 部屋全体の温度管理 | 風が直接当たらないようにする |

## 季節の変わり目の注意

春と秋の季節の変わり目は気温の変動が大きく、体調を崩しやすい時期です。

- この時期は、元気の様子と排泄を普段より頻繁に観察します。
- 必要に応じて窓を開ける時間を減らし、隙間風を避けます。
- 換羽期は体力を多く使うため、栄養を適度に補います。
- 朝晩の寒暖差に注意し、夜は早めにケージカバーを掛けます。

## 梅雨時の湿気対策

- 湿度が長期間70％を超えると、カビや呼吸器の問題が起こりやすくなります。
- 除湿器またはエアコンの除湿機能を使います。
- カビを防ぐため、ケージ底の敷紙をこまめに替えます。
- 餌は密閉して保管し、湿気による変質を防ぎます。

> ヒント：記録機能付きの温湿度計を用意すると、一日の最高値と最低値を後から確認でき、危険な変化を見つけやすくなります。
