# ケージの日常清掃と消毒

清潔なケージは、呼吸器や消化器の病気を防ぐ基本です。「毎日、餌と水を替えれば十分」と考える初心者も多いですが、フン、羽粉、食べかすには細菌がすぐに繁殖します。ここでは、「毎日・毎週・毎月」の3段階に分けて清掃のペースを説明します。

## 毎日必ず行うこと

朝晩に一度ずつ確認することをおすすめします。

- **水を替える**：細菌によるぬめりを防ぐため、水入れを毎日しっかり洗います。夏は1日2回替えてもかまいません。
- **餌入れを掃除する**：殻や細かなくずを捨て、新鮮な餌を補充します。
- **止まり木を拭く**：ウェットシートや古い歯ブラシでフンを取り、細菌の蓄積を防ぎます。
- **敷紙・敷材を替える**：底トレーの敷紙は毎日交換し、底を乾燥した状態に保ちます。
- **おもちゃを確認する**：かじって壊れたおもちゃは早めに交換し、破片の誤飲を防ぎます。

## 毎週の念入りな清掃

週に一日、インコを予備のケージまたはキャリーへ移してから、次の作業を行います。

1. 餌入れ、水入れ、止まり木、おもちゃを取り外します。
2. ぬるい石けん水で洗い、よくすすいで乾かします。
3. 鳥用の消毒液でケージ本体と底トレーを拭きます。
4. すべての部品が完全に乾いてから元に戻します。
5. 同時に、ケージにさびや溶接部の割れがないか確認します。

## 毎月のメンテナンス

- ケージの隙間にたまった汚れまで徹底的に洗い流します。
- 止まり木の表面がひどく摩耗していないか確認し、必要に応じて研磨または交換します。
- ケージカバーを洗うか交換します。
- 餌や薬の使用期限を確認します。

## 消毒剤の選び方

| 推奨 | 非推奨 |
|---|---|
| F10消毒剤 | 塩素系漂白剤 |
| 希釈した次亜塩素酸水 | アルコールの直接噴霧 |
| 鳥専用消毒剤 | フェノール系洗剤 |
| 高温スチーム | 強酸・強アルカリ洗剤 |

消毒後は必ず清水でもう一度拭き、窓を開けて30分以上換気してください。臭いが完全になくなってからインコを戻します。

## 清掃手順のコツ

- 上から下へ：二次汚染を防ぐため、先に止まり木とおもちゃを拭き、最後に底トレーを清掃します。
- 道具を分ける：交差汚染を防ぐため、ケージ用の布と餌入れ用の布を分けます。
- 餌入れと水入れを2組用意する：一組を使用し、もう一組を乾燥させ、毎日交互に使います。

> ヒント：香りの強い洗剤は使わないでください。インコの呼吸器は非常に敏感です。また、レモンやティーツリーなどの「天然」精油も鳥には有毒です。
