# 笼舍日常清洁与消毒

干净的笼舍是预防呼吸道和消化道疾病的基础。很多新手以为“每天换食换水就够了”，其实粪便、羽粉、食物残渣都会快速滋生细菌。

## 每日必做

建议早晚各检查一次：

- **换水**：水盒每天彻底清洗，避免细菌膜。
- **清理食盒**：倒掉空壳和碎渣，补充新鲜食物。
- **擦站杆**：用湿巾或旧牙刷清理粪便，防止细菌堆积。
- **换垫纸/垫料**：底盘垫纸每天更换。

## 每周深度清洁

每周选一天把鹦鹉转移到备用笼或外出笼，然后：

1. 拆下食盒、水盒、站杆、玩具。
2. 用温热肥皂水刷洗，冲净后晾干。
3. 用鹦鹉专用消毒液擦拭笼身和底盘。
4. 等所有部件干透后再装回。

## 消毒剂怎么选

| 推荐 | 不推荐 |
|---|---|
| F10 消毒液 | 84 消毒液 |
| 稀释次氯酸 | 酒精直接喷洒 |
| 鹦鹉专用消毒液 | 酚类清洁剂 |

消毒后务必用清水再擦一遍，并开窗通风 30 分钟以上。

> 小贴士：不要用香味过重的清洁剂，鹦鹉的呼吸道非常敏感。
